# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 프로젝트 비전 (What & Why)
   - 핵심 아이디어 (한 줄 요약)
   - 문제 정의 (3~5개)
   - 해결책
   - 개발 철학

2. 브랜드 정체성
   - 브랜드 의미
   - 슬로건
   - 테마

3. 핵심 기능 (Core Features)
   - 주요 기능별 상세 설명
   - 화면 위치 및 동작 흐름

4. 기술 스택/구조
   - 프론트엔드 / 백엔드 / 배포 / AI 연동
   - 프로젝트 구조

---

## A3. AI 작업 순서 (5단계)

### 1단계: Order Sheet 완전 이해

**체크리스트**:
- [ ] A2 작업 내용 확인
- [ ] A4 산출물 확인
- [ ] PART_B 특별 지시사항/참고사항 확인

**출력**: `'Order Sheet 확인 완료. 단계: P2-1'`

---

### 2단계: 문의사항 질문

**질문 형식**:
```
[P2-1] 질문: {내용}
옵션 A: {옵션1}
옵션 B: {옵션2}
```

**출력**: 질문 목록 또는 `'질문 없음'`

---

### 3단계: 실행 (Execution)

**체크리스트**:
- [ ] 프로젝트 비전 작성
- [ ] 브랜드 정체성 정의
- [ ] 핵심 기능 정리
- [ ] 기술 스택/구조 작성

---

### 4단계: 검증 (Verification)

**체크리스트**:
- [ ] 프로젝트 비전이 명확한가?
- [ ] 핵심 기능이 구체적인가?
- [ ] 기술 스택이 적절한가?

**출력**: `'검증 완료'`

---
