# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 기능 요구사항 정의
   - 핵심 기능 목록
   - 기능별 상세 설명
   - 입력/출력 정의

2. 비기능 요구사항 정의
   - 성능 요구사항
   - 보안 요구사항
   - 사용성 요구사항

3. 우선순위 분류
   - Must Have (필수)
   - Should Have (권장)
   - Nice to Have (선택)

---

## A3. AI 작업 순서 (5단계)

### 1단계: Order Sheet 완전 이해

---

### 2단계: 문의사항 질문


**출력**: 질문 목록 또는 `'질문 없음'`

- [ ] 비기능 요구사항 정의
