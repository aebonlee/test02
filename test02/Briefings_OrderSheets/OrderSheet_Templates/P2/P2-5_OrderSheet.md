# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 컬러 시스템 정의
   - Primary/Secondary 컬러
   - Semantic 컬러 (성공/경고/오류)
   - 다크 모드 대응 (선택)

2. 타이포그래피 정의
   - 폰트 패밀리 선정
   - 폰트 사이즈 체계
   - 라인 높이 및 자간

3. 컴포넌트 스타일 정의
   - 버튼 스타일
   - 입력 필드 스타일
   - 카드, 모달 등

---

## A3. AI 작업 순서 (5단계)

### 1단계: Order Sheet 완전 이해

**체크리스트**:
- [ ] A2 작업 내용 확인
- [ ] A4 산출물 확인
- [ ] PART_B 특별 지시사항/참고사항 확인

**출력**: `'Order Sheet 확인 완료. 단계: P2-5'`

---

### 2단계: 문의사항 질문

**질문 형식**:
```
[P2-5] 질문: {내용}
옵션 A: {옵션1}
옵션 B: {옵션2}
```

**출력**: 질문 목록 또는 `'질문 없음'`

---

### 3단계: 실행 (Execution)

**체크리스트**:
- [ ] 컬러 시스템 정의
- [ ] 타이포그래피 정의
- [ ] 컴포넌트 스타일 정의

---

### 4단계: 검증 (Verification)

**체크리스트**:
- [ ] 컬러 시스템이 일관성 있는가?
- [ ] 웹 접근성 기준을 충족하는가?
- [ ] 확장 가능한 시스템인가?

**출력**: `'검증 완료'`

---

### 5단계: 완료 보고 (Report)

**보고서 생성**:
- 파일명: `P2-5_completion_report.md`
- 저장 위치: `Human_ClaudeCode_Bridge/Reports/`
