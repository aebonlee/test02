# Order Sheet - Business_Model

> **버전**: 5.4
> **단계**: P1-3 (비즈니스 모델)
> **목적**: 수익 창출 방법과 비즈니스 구조 정의

---

# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 비즈니스 모델 캔버스 작성
   - 고객 세그먼트
   - 가치 제안
   - 채널
   - 고객 관계
   - 수익원
   - 핵심 자원
   - 핵심 활동
   - 핵심 파트너
   - 비용 구조

2. 수익 모델 정의
   - 주요 수익원 (구독료, 광고, 수수료 등)
   - 가격 정책
   - 예상 매출 시나리오

3. 비용 구조 분석
   - 고정 비용
   - 변동 비용
   - 초기 투자 비용

4. 손익 분기점 분석
   - BEP (Break-Even Point) 계산
   - 목표 달성 시나리오

---

## A3. AI 작업 순서 (5단계)

### 1단계: Order Sheet 완전 이해

**체크리스트**:
- [ ] A2 작업 내용 확인
- [ ] A4 산출물 확인
- [ ] PART_B 특별 지시사항/참고사항 확인

**출력**: `'Order Sheet 확인 완료. 단계: P1-3'`

---

### 2단계: 문의사항 질문

**질문 형식**:
```
[P1-3] 질문: {내용}
옵션 A: {옵션1}
옵션 B: {옵션2}
```

**출력**: 질문 목록 또는 `'질문 없음'`

---

### 3단계: 실행 (Execution)

**체크리스트**:
- [ ] 비즈니스 모델 캔버스 작성
- [ ] 수익 모델 정의
- [ ] 비용 구조 분석
- [ ] BEP 분석

---

### 4단계: 검증 (Verification)

**체크리스트**:
- [ ] 비즈니스 모델 캔버스가 완성되었는가?
- [ ] 수익 모델이 실현 가능한가?
- [ ] 비용 구조가 현실적인가?
- [ ] BEP 분석이 합리적인가?

**출력**: `'검증 완료'`
