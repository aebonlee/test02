> **목적**: 완료된 프로젝트에 수정 작업 요청

---

# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 수정 요청 내용을 정확히 이해할 때까지 작업 시작 금지
2. 수정 범위 외의 코드 변경 금지
3. 수정 전 영향받는 파일 백업 권장
4. 수정 후 테스트 결과 보고 필수
5. 거짓 기록 절대 금지

- [ ] 콘텐츠 수정
- [ ] 기타: _______________

## A3. AI 작업 순서 (4단계)

### 1단계: 요청 확인
- 수정 내용 파악
- 영향 범위 분석
- 관련 파일 식별

**출력**: `'수정 요청 확인 완료. 영향 파일: [파일 목록]'`

---

### 2단계: 수정 실행
- 관련 파일 수정
- 코드 변경사항 적용
- 변경 전/후 비교

---

### 3단계: 검증
- 수정 사항 테스트
- 기존 기능 영향 확인
- 에러 발생 여부 확인

**출력**: `'검증 완료'`

---

### 4단계: 완료 보고

**보고 내용:**
- 수정 내역 요약
- 변경된 파일 목록
- 테스트 결과
- 주의사항 (있을 경우)
