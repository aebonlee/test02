# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. README 작성
   - 프로젝트 소개
   - 설치/실행 방법
   - 주요 기능 설명

2. 구조 문서
   - 폴더/파일 구조 설명
   - 아키텍처 개요

3. 가이드 문서
   - 빌드/배포 가이드
   - 트러블슈팅

---

## A3. AI 작업 순서 (5단계)

### 1단계: Order Sheet 완전 이해

**체크리스트**:
- [ ] A2 작업 내용 확인
- [ ] A4 산출물 확인
- [ ] PART_B 특별 지시사항/참고사항 확인

**출력**: `'Order Sheet 확인 완료. 단계: P3-3'`

---

### 2단계: 문의사항 질문

**질문 형식**:
```
[P3-3] 질문: {내용}
옵션 A: {옵션1}
옵션 B: {옵션2}
```

**출력**: 질문 목록 또는 `'질문 없음'`

---

### 3단계: 실행 (Execution)

**체크리스트**:
- [ ] README 작성
- [ ] 구조 문서 작성
- [ ] 가이드 문서 작성

---

### 4단계: 검증 (Verification)

**체크리스트**:
- [ ] 문서가 명확하고 이해하기 쉬운가?
- [ ] 필요한 정보가 모두 포함되었는가?
- [ ] 실제 코드/구조와 일치하는가?

**출력**: `'검증 완료'`

---

### 5단계: 완료 보고 (Report)

**보고서 생성**:
- 파일명: `P3-3_completion_report.md`
- 저장 위치: `Human_ClaudeCode_Bridge/Reports/`

**보고 내용**:
- 완료된 작업 요약
- 생성된 문서 목록
- 다음 단계 안내 (S0)
