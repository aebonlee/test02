# Service_Introduction

## 단계 개요

**서비스를 소개하는 문서**를 작성합니다.

외부에 서비스를 설명하거나, 랜딩 페이지에 사용할 콘텐츠를 준비하는 단계입니다.

## 필요성

- **마케팅 준비**: 서비스 출시 전 홍보 자료를 준비합니다
- **메시지 통일**: 일관된 메시지로 서비스를 설명합니다
- **가치 전달**: 사용자에게 서비스의 가치를 명확히 전달합니다

## 작성 내용

### 서비스 소개
- 한 줄 소개
- 핵심 가치 제안
- 주요 기능 설명

### 대상 고객
- 타겟 사용자
- 사용자 페르소나
- 사용 시나리오

### 차별화 포인트
- 경쟁 서비스 대비 장점
- 독특한 기능
- 기술적 강점

## PO가 결정할 것

- 서비스 핵심 메시지
- 타겟 고객
- 강조할 기능
