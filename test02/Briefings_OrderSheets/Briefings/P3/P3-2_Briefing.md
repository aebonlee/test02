# Database

## 단계 개요

**데이터베이스를 실제로 생성**합니다.

P2-4에서 설계한 구조를 바탕으로 실제 데이터베이스를 만드는 단계입니다.

## 필요성

- **데이터 저장**: 서비스에서 사용할 데이터를 저장할 곳을 만듭니다
- **보안 적용**: 누가 어떤 데이터에 접근할 수 있는지 설정합니다
- **테스트 환경**: 프로토타입 테스트에 필요한 데이터를 넣습니다

## 무엇을 만드나요?

### 데이터베이스 구축

- 테이블 생성: 데이터 저장 공간
- 보안 정책: 접근 권한 설정
- 초기 데이터: 테스트용 샘플 데이터

### 보안 설정

- 행 수준 보안: 사용자별 데이터 접근 제한
- 역할 권한: 인증된 사용자만 접근

## PO가 할 일

- 테이블 확인: 필요한 테이블 목록 확인
- DB 선택: Supabase 등 DB 서비스 선택
- 초기 데이터: 테스트에 필요한 샘플 데이터
