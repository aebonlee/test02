# Frontend

## 단계 개요

**프론트엔드 프로토타입**을 제작합니다.

UI/UX 디자인을 바탕으로 실제 화면을 구현하는 단계입니다.

## 필요성

- **시각화**: 설계한 화면을 실제로 볼 수 있습니다
- **검증**: 사용성을 미리 테스트합니다
- **소통**: 개발 방향을 명확히 공유합니다

## 무엇을 만드나요?

### 페이지 종류

- 홈/랜딩: 메인 페이지
- 인증: 로그인, 회원가입
- 대시보드: 사용자 메인 화면
- 설정: 계정 설정

### 구현 항목

- 레이아웃: 페이지별 구성
- 폼 요소: 입력 및 버튼
- 모달: 대화상자
- 상태 표시: 로딩, 오류 등

## PO가 결정할 것

- 필요한 페이지 목록
- 페이지별 우선순위
- 각 페이지의 핵심 기능
