# 완료된 프로젝트 수정

## 기능 개요

완료된 프로젝트를 다시 열어 **수정 작업**을 진행합니다.

버그 수정, 기능 개선, UI 변경 등 완료된 프로젝트에 추가 작업이 필요할 때 사용합니다.

## 사용 시점

- 버그 수정: 발견된 오류나 문제 해결
- 기능 추가: 새로운 기능 추가
- 기능 개선: 기존 기능 성능/UX 개선
- UI 수정: 디자인, 레이아웃 변경
- 콘텐츠 수정: 텍스트, 이미지 등 수정

## 수정 작업 흐름

1. 수정할 내용 명확히 정리
2. Order Sheet에 수정할 사항 정리
3. Claude Code에게 Order Sheet 전달
4. 수정 결과 확인 및 테스트
5. 수정 완료 후 재배포


> 위의 수정작업을 지시하는 Order Sheet를 Control Desk에 로딩하시겠습니까?
