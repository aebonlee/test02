# Dev Package와 도구 설치 및 프로젝트 초기화 안내

---

## 1. Package 다운로드

**경로:**
```
My Page > 자료 다운로드 > "Dev Package 다운로드" 카드 > "Dev Package 다운로드 (ZIP)" 버튼 클릭
```

**다운로드 후:**
1. ZIP 압축 해제
2. 원하는 위치로 폴더 이동
3. 폴더명 변경 (프로젝트에 맞게)

---

## 2. 도구 설치

**Package만 다운로드 받으면 끝이 아닙니다.**
Package는 폴더 구조와 템플릿만 들어있어서, 개발 도구를 따로 설치해야 합니다.

**설치할 도구 7가지:**

1. **Claude Code** - 터미널에서 `npm install -g @anthropic-ai/claude-code` 입력 (AI 개발 어시스턴트)
2. **Node.js** - https://nodejs.org 에서 다운로드 (LTS 버전 권장)
3. **Git** - https://git-scm.com 에서 다운로드
4. **VS Code** - https://code.visualstudio.com 에서 다운로드
5. **GitHub CLI** - https://cli.github.com 에서 다운로드
6. **Vercel CLI** - 터미널에서 `npm install -g vercel` 입력
7. **Supabase** - https://supabase.com 에서 가입 후 프로젝트 생성

---

## 3. 프로젝트 초기화

**도구 설치만 하면 끝이 아닙니다.**
Package 폴더에서 초기화 작업까지 마쳐야 개발을 시작할 수 있습니다.

**Package 폴더란?**
1단계를 완료하면 내 컴퓨터에 프로젝트 폴더가 생깁니다. 이 폴더가 Package 폴더입니다.

**초기화 방법:**
1. Package 폴더에서 터미널을 엽니다 (폴더 우클릭 > 터미널에서 열기)
2. `claude` 입력 후 엔터
3. Claude Code가 실행되면 이렇게 요청하세요:

```
"프로젝트 초기화해 줘. git init, npm init 하고, .gitignore 파일 만들고, 필요한 npm 패키지 설치해 줘."
```

---

**3단계 모두 완료 후 개발을 시작할 수 있습니다.**
