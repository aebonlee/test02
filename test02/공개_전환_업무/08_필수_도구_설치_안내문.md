## 1. Git (필수)

**Git이 하는 일:**

✅ 모든 작업 내역을 자동 저장 (언제든 되돌리기 가능)

✅ 커밋과 동시에 자동 실행 (Pre-commit Hook):
   1. Order Sheets MD → JS 파일로 번들링
   2. Briefings (상황별 안내문) MD → JS 파일로 번들링
   3. 외부 연동 설정 가이드 MD → JS 파일로 번들링
   4. 서비스 소개 모달 MD → index.html에 삽입
   5. SAL Grid 매뉴얼 MD → HTML 변환
   6. 빌더 계정 매뉴얼 MD → HTML 변환
   7. P0~S5 진행률 → JSON 파일 생성
   8. Stage 폴더 → 배포 폴더로 자동 복사

**Git 없이는:**

❌ 실수하면 복구 불가

❌ 위 8가지 자동화 전부 안 됨 (매번 수동으로 해야 함)

❌ 작업 이력 추적 불가

### 다운로드

| OS | 다운로드 링크 |
|----|--------------|
| **Windows** | https://git-scm.com/download/win |
| **Mac** | https://git-scm.com/download/mac |
| **Linux** | https://git-scm.com/download/linux |

### 설치 확인

```bash
git --version
```

## 4. Claude Code

AI 개발 어시스턴트입니다.

### 설치 명령어

터미널(명령 프롬프트)에서 실행:

```bash
npm install -g @anthropic-ai/claude-code
```

### 설치 확인

```bash
claude --version
```

### API 키 설정

1. https://console.anthropic.com/ 접속
2. API Keys 메뉴에서 키 생성
3. 첫 실행 시 키 입력 또는 환경변수 설정:

```bash
# Windows (PowerShell)
$env:ANTHROPIC_API_KEY="your-api-key"

# Mac/Linux
export ANTHROPIC_API_KEY="your-api-key"
```

## 7. Supabase

데이터베이스 서비스입니다.

### 가입 및 프로젝트 생성

1. https://supabase.com 접속
2. 회원가입 (GitHub 계정으로 가능)
3. New Project 생성
4. 프로젝트 Settings > API에서 키 확인

### 필요한 키

| 키 | 용도 |
|----|------|
| **Project URL** | 프로젝트 주소 |
| **anon public** | 클라이언트용 공개 키 |
| **service_role** | 서버용 비밀 키 (노출 금지) |

### 환경변수 설정

프로젝트 루트에 `.env` 파일 생성:

```
SUPABASE_URL=your-project-url
SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
```

### Supabase CLI (선택)

```bash
npm install -g supabase
```

```bash
supabase --version
```

## 8. 프로젝트 초기화

패키지 폴더에서 개발에 필요한 기본 파일들을 생성합니다.

### 초기화 명령어

패키지 폴더로 이동 후 순서대로 실행:

```bash
# 1. Git 저장소 초기화
git init

# 2. npm 프로젝트 초기화 (package.json 생성)
npm init -y

# 3. Claude Code 실행
claude
```

### Claude Code에게 요청할 것

```
"프로젝트 초기화해 줘. .gitignore 파일 만들고, 필요한 npm 패키지 설치해 줘."
```

### 초기화 시 생성되는 파일

| 파일 | 생성 방법 | 용도 |
|------|----------|------|
| `.git/` | `git init` | Git 버전 관리 |
| `package.json` | `npm init -y` | npm 프로젝트 설정 |
| `.gitignore` | Claude Code 요청 | Git 제외 파일 목록 |
| `node_modules/` | `npm install` | 설치된 패키지들 |

### 개발하면서 생성하는 파일

| 파일 | 생성 시점 | 용도 |
|------|----------|------|
| `index.html` | 프론트엔드 개발 시작 | 메인 페이지 |
| `vercel.json` | 배포 설정 시 | Vercel 배포 설정 |
| `.env` | 외부 서비스 연동 시 | 환경변수 (API 키 등) |

## 문제 해결

### "command not found" 오류

- 터미널을 껐다가 다시 열어보세요
- 설치 후 시스템 재시작이 필요할 수 있습니다

### npm 권한 오류 (Mac/Linux)

```bash
sudo npm install -g @anthropic-ai/claude-code
```

### Windows에서 PowerShell 실행 정책 오류

관리자 권한으로 PowerShell 실행 후:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
