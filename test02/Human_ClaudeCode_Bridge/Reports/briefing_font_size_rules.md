# Briefing 글자 크기 규칙

> 생성일: 2026-01-04
> 적용 파일: `Briefings_OrderSheets/Briefings/generate-briefings-js.js`

---

## 글자 크기 기준

| 레벨 | 마크다운 | 크기 | 스타일 | 용도 |
|------|----------|------|--------|------|
| h1 | `#` | 15px | 굵게 (700) | 메인 제목 (1개) |
| h2 | `##` | 14px | 굵게 (600) | 섹션 제목 |
| h3 | `###` | 13px | 굵게 (600) | 소섹션 제목 (거의 안 씀) |
| 볼드 | `**텍스트**` | 12px | 굵게 | 강조/소제목 |
| 본문 | 일반 텍스트 | 12px | 보통 | 본문 내용 |
| 표 | `\| \|` | 12px | 보통 | 테이블 내용 |
| 리스트 | `- 항목` | 12px | 보통 | 목록 항목 |

---

## 핵심 원칙

1. **사이드바와 통일**: 사이드바 글자 크기(11-13px)와 맞춤
2. **h3 대신 볼드 사용**: h3(13px)보다 `**볼드**`(12px)로 구분 권장
3. **본문 통일**: 표, 리스트, 단락 모두 12px

---

## Default_v2.md 적용 예시

```
# SSAL Works에 오신 것을 환영합니다     ← h1 (15px)

## 먼저 둘러보세요                      ← h2 (14px)

**SSAL Works의 10가지 차별점**          ← 볼드 (12px, 굵게)

- 왼쪽 사이드바: 10단계 진행 프로세스    ← 리스트 (12px)

일반 설명 텍스트입니다.                  ← 본문 (12px)
```

---

## generate-briefings-js.js 관련 코드

```javascript
// 제목 (h1=15px, h2=14px, h3=13px)
html = html.replace(/^# (.+)$/gm, '<h1 style="font-size: 15px; ...">$1</h1>');
html = html.replace(/^## (.+)$/gm, '<h2 style="font-size: 14px; ...">$1</h2>');
html = html.replace(/^### (.+)$/gm, '<h3 style="font-size: 13px; ...">$1</h3>');

// 본문/표/리스트 (모두 12px)
// 표: font-size: 12px
// 단락: font-size: 12px
// 리스트: font-size: 12px
```
