## 공식 명칭: 데이터 로컬 저장 방식

SSAL Works는 **데이터 로컬 저장 방식**을 채택합니다.

모든 프로젝트 자료가 서비스 회사 서버가 아닌 **내 컴퓨터에 저장**되어,
서비스 종료와 관계없이 내 결과물은 영원히 내 것입니다.

---

## 다른 서비스와 뭐가 다른가요?

보통 온라인 서비스를 이용하면, 내가 만든 자료들이 어디에 저장될까요?

**일반적인 서비스의 경우:**
- 내가 작성한 문서, 코드, 데이터 → **서비스 회사 서버**에 저장
- 서비스 종료하면? → 자료 날아갈 수 있음
- 구독 취소하면? → 접근 불가능해질 수 있음
- 다른 곳으로 옮기고 싶으면? → 복잡한 내보내기 과정 필요

**SSAL Works는 다릅니다 (데이터 로컬 저장 방식):**
- 내가 작성한 문서, 코드, 데이터 → **내 컴퓨터에 저장**
- 서비스 종료해도 → 내 컴퓨터에 그대로 있음
- 구독 취소해도 → 파일은 여전히 내 것
- 다른 곳으로 옮기고 싶으면 → 폴더 복사하면 끝

---

## 쉽게 비유하면

**일반 서비스** = 서비스 회사 창고에 내 물건 보관
- 월세 내면 사용 가능
- 월세 안 내면 물건 꺼내기 어려움
- 서비스 회사가 망하면 물건 잃어버릴 수도

**SSAL Works (데이터 로컬 저장 방식)** = 내 집에 물건 보관, 전문가가 정리 도와줌
- 전문가 도움받아 깔끔하게 정리
- 도움 안 받아도 물건은 내 집에 그대로
- 언제든 내 마음대로 사용 가능

---

## 실제로 어떻게 되나요?

SSAL Works를 시작하면 "Dev Package"라는 폴더를 다운로드합니다.

이 폴더 안에는:
- 📁 프로젝트 기획 문서
- 📁 개발 코드
- 📁 진행 상황 기록
- 📁 AI와 협업한 내용

이 모든 것이 **내 컴퓨터**에 저장됩니다.

```
내 컴퓨터
└── 내프로젝트_폴더/
    ├── 기획 문서들
    ├── 개발 코드들
    ├── 진행 기록들
    └── ... 모든 자료
```

서비스 회사 서버 어딘가가 아니라, 지금 내 눈앞에 있는 컴퓨터에요.

---

## 이게 왜 중요한가요?

### 1. 진짜 내 것
- 내가 만든 건 내 것입니다
- 누구도 뺏어갈 수 없습니다

### 2. 영원히 보존
- 10년 후에도 폴더 열면 그대로 있습니다
- 서비스 회사가 어떻게 되든 상관없습니다

### 3. 자유롭게 활용
- 다른 도구로 열어볼 수 있습니다
- 다른 사람에게 공유할 수 있습니다
- 백업도 마음대로 할 수 있습니다

### 4. 인터넷 없이도 OK
- 파일이 내 컴퓨터에 있으니까요
- 오프라인에서도 확인 가능합니다

## 유일하게 수집하는 정보: 진행률

SSAL Works는 **데이터 로컬 저장 방식**을 채택하지만, 딱 하나 수집하는 정보가 있습니다.
바로 **진행률**입니다.

### 수집하는 정보

| 정보 | 내용 |
|------|------|
| 프로젝트 시작일 | 언제 시작했는지 |
| 각 단계 완료일 | P0, P1, S1, S2... 언제 끝났는지 |
| 프로젝트 종료일 | 언제 완료했는지 |
| 소요 기간 | 총 몇 개월 걸렸는지 |

### 수집하지 않는 정보

- 파일 내용
- 코드 내용
- 기획 문서 내용
- 어떤 프로젝트인지
- 무슨 사업인지

**즉, "얼마나 진행됐는지"만 알고, "무엇을 만드는지"는 전혀 모릅니다.**

### 진행률 수집 목적: 이용자 혜택 제공

| 용도 | 설명 |
|------|------|
| 완료 여부 확인 | 프로젝트가 끝났는지 판단 |
| 소요 기간 파악 | 몇 개월 걸렸는지 계산 |
| 빠른 완료 혜택 | 신속하게 완료하면 베네핏 제공 |
| 얼리버드 환불 조건 | 30명 고객 확보 시 환불 자격 검증 |

### 예시: 얼리버드 프로그램

얼리버드 이용자가 프로젝트를 완료하고 고객 30명을 확보하면 환불 혜택이 있습니다.
이 조건 충족 여부를 확인하려면 "프로젝트가 끝났는지"를 알아야 합니다.
그래서 진행률 데이터가 필요한 것입니다.

---

## 요약: 데이터 로컬 저장 방식 + 진행률만 수집

| 구분 | 저장 위치 | 수집 여부 |
|------|----------|----------|
| 프로젝트 파일 (코드, 문서 등) | 내 컴퓨터 | ❌ 수집 안 함 |
| 진행률 (몇 단계 완료했는지) | 서버 | ✅ 수집함 (혜택 제공 목적) |

> **핵심:**
> 내용은 절대 수집하지 않습니다.
> 오직 "얼마나 진행됐는지"만 파악해서 이용자 혜택을 제공합니다.
