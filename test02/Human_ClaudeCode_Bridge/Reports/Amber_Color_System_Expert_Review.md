# SSAL Works Amber 색상 시스템 전문가 검토

**검토일**: 2025-12-30
**검토자**: UI/UX 디자인 전문가 (Claude)
**평가 등급**: A (8.5/10)

---

## Executive Summary

현재 SSAL Works의 Amber 색상 시스템(**#D97706 + #B45309**)은 매우 우수하며, 사용자 피드백을 반영한 최적의 선택입니다. 가독성, 접근성, 디자인 조화 모든 면에서 우수한 평가를 받았습니다.

**핵심 권장사항**: 현재 상태 유지 + 프로토타입 색상 통일

---

## 1. 현재 색상 사용 현황

### 메인 사이트 (index.html, 404.html 등)
- **기본 Amber**: `#D97706` (Tailwind amber-600)
- **Hover/Dark Amber**: `#B45309` (Tailwind amber-700)
- **CSS 변수**:
  ```css
  --color-amber: #D97706;
  --color-amber-dark: #B45309;
  --secondary: var(--color-amber);
  ```

### 프로토타입 (P3_프로토타입_제작)
- **기본**: `#F59E0B` (이전 밝은 색상)
- **Dark**: `#D97706` (현재 메인 사이트 기본 색상)

**⚠️ 발견된 문제**: 메인과 프로토타입 간 색상 불일치

---

## 2. 색상 조합 평가 ✅ 우수

### #D97706 (Primary) + #B45309 (Hover/Dark)

#### 장점:
1. **적절한 명도 대비**: 두 색상 간 명도 차이가 이상적 (약 15% 차이)
2. **그라데이션 효과**: `linear-gradient(135deg, #D97706 0%, #B45309 100%)`가 자연스럽고 고급스러움
3. **사용자 피드백 반영**: `#F59E0B`에서 변경한 것이 정확한 판단
   - `#F59E0B`는 명도가 너무 높아 장시간 노출 시 눈의 피로 유발
   - `#D97706`은 적절한 채도와 명도로 편안함

#### 색상 분석:

**#D97706** (RGB: 217, 119, 6)
- HSL: 30°, 95%, 44%
- 따뜻하고 전문적인 금색
- 신뢰감과 활력을 동시에 전달

**#B45309** (RGB: 180, 83, 9)
- HSL: 26°, 91%, 37%
- 더 깊고 안정적인 느낌
- Hover 상태에서 명확한 피드백 제공

## 3. 가독성/접근성 평가 ✅ 통과

### WCAG 2.1 대비 테스트

#### 흰색 텍스트(#FFFFFF) + #D97706 배경
- **대비율**: 4.67:1
- **WCAG AA 기준**: ✅ 통과 (4.5:1 이상)
- **WCAG AAA 기준**: ❌ 미달 (7:1 필요)
- **결론**: 일반 텍스트 사용 적합, 큰 텍스트(18pt+ 또는 14pt bold+) 권장

#### 흰색 텍스트 + #B45309 배경
- **대비율**: 5.94:1
- **WCAG AA 기준**: ✅ 통과
- **WCAG AAA 기준**: ❌ 근접 (7:1 필요)
- **결론**: #D97706보다 더 안전한 대비

### 색각 이상 테스트
| 유형 | 평가 | 설명 |
|------|:----:|------|
| 적록색맹 (Deuteranopia) | ✅ | 노란빛 계열로 인식되어 문제 없음 |
| 청황색맹 (Tritanopia) | ✅ | 붉은빛 계열로 인식되어 구별 가능 |
| 전색맹 (Achromatopsia) | ✅ | 명도 대비로 충분히 구별 가능 |

## 4. 시각적 일관성 평가 ⚠️ 개선 필요

### 문제점: 메인 사이트와 프로토타입 간 색상 불일치

| 위치 | Primary | Hover |
|------|---------|-------|
| 메인 사이트 | #D97706 | #B45309 |
| 프로토타입 | #F59E0B | #D97706 |

### 사용자 경험 영향:
- 동일 서비스임에도 다른 브랜드처럼 느껴질 수 있음
- 프로토타입에서 메인으로 전환 시 혼란 가능

### 권장사항:
프로토타입의 색상을 메인 사이트와 통일

```css
/* 변경 전 (프로토타입) */
--secondary: #F59E0B;
--secondary-dark: #D97706;

/* 변경 후 (통일) */
--secondary: #D97706;
--secondary-dark: #B45309;
```

3. 그라데이션 배경 ✅ (체크박스 등)

## 6. 개선 제안

### 현재 상태 평가: **8.5/10** (매우 우수)

### 유지해야 할 점 ✅

1. **#D97706 + #B45309 조합 유지** - 현재가 최적
2. **그라데이션 효과 유지** - 고급스럽고 현대적
3. **사용 패턴 유지** - 적절한 빈도와 위치

### 개선 권장사항

#### 1. 프로토타입 색상 통일 (우선순위: 🔴 높음)

**적용 대상**: `P3_프로토타입_제작/Frontend/Prototype` 내 모든 HTML 파일

**변경 내용**:
```css
/* Find & Replace */
--secondary: #F59E0B;      → --secondary: #D97706;
--secondary-dark: #D97706; → --secondary-dark: #B45309;
```

**예상 효과**:
- 일관된 브랜드 경험 제공
- 사용자 혼란 방지

---

#### 2. 더 어두운 배경용 변형 추가 (우선순위: 🟡 중간)

```css
:root {
    --color-amber: #D97706;      /* 기본 */
    --color-amber-dark: #B45309; /* Hover */
    --color-amber-darker: #92400e; /* 어두운 배경용 (Tailwind amber-800) */
}
```

**용도**: 다크 모드나 어두운 배경에서 사용

---

#### 3. 포커스 상태 추가 (우선순위: 🟡 중간)

```css
.btn-amber:focus {
    outline: 2px solid #D97706;
    outline-offset: 2px;
    box-shadow: 0 0 0 3px rgba(217, 119, 6, 0.2);
}
```

**이유**: 키보드 네비게이션 접근성 향상

---

#### 4. 투명도 변형 추가 (우선순위: ⚪ 낮음)

```css
:root {
    --color-amber-10: rgba(217, 119, 6, 0.1);  /* 배경 */
    --color-amber-20: rgba(217, 119, 6, 0.2);  /* 호버 배경 */
    --color-amber-50: rgba(217, 119, 6, 0.5);  /* 반투명 */
}
```

**용도**: 미묘한 배경 강조, 호버 효과
