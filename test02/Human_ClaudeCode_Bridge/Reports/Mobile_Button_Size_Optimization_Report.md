## 1. 문제 분석

- 모바일에서 버튼 높이가 업계 평균보다 **8-12px 큼**
- 사용자 피드백: "버튼이 너무 크게 보인다"

|--------|------|--------|----------|------|
| 홈 | 검색 버튼 | 52px | 44px | ⚠️ 큼 |
| 커뮤니티 | 검색 버튼 | 52px | 44px | ⚠️ 큼 |
| 커뮤니티 | 탭 버튼 | 48px | 40px | ⚠️ 약간 큼 |
| 커뮤니티 | 글쓰기 버튼 | 48px | 44px | ⚠️ 약간 큼 |
| 정치인 | 검색 버튼 | 52px | 44px | ⚠️ 큼 |
| 정치인 | 필터 버튼 | 52px | 44px | ⚠️ 큼 |

| 앱 | 주요 버튼 높이 | 특징 |
|----|---------------|------|

## 2. 수정 내용

### 원인
```css
/* 기존 */
py-3 = 12px (상) + 12px (하) = 24px 패딩
+ 텍스트 높이 (~16px) + border
= 약 52px
```

### 해결책
```css
/* 수정 */
py-2.5 = 10px (상) + 10px (하) = 20px 패딩
py-2 = 8px (상) + 8px (하) = 16px 패딩
= 약 44-48px
```
