# SSALWorks 경쟁사 분석 (Competitor Analysis)

> **최종 업데이트:** 2025-11-28
> **작성자:** Sunny (써니)
> **버전:** 1.0

---

## 핵심 결론: 직접적인 경쟁자는 없다

SSALWorks는 **도구 + 교육 + 성공 샘플 + 전문가 자문**을 하나로 결합한 플랫폼입니다.

이러한 **토탈 서비스**를 제공하는 직접적인 경쟁자는 현재 존재하지 않습니다.

---

## 유사 서비스 비교

### 1. 웹사이트 빌더

| 서비스 | 특징 | SSALWorks와의 차이 |
|--------|------|-------------------|
| **Wix** | 드래그앤드롭 웹사이트 빌더 | 템플릿 기반, 커스터마이징 한계 |
| **Squarespace** | 디자인 중심 웹사이트 빌더 | 템플릿 의존, 복잡한 기능 구현 어려움 |
| **Webflow** | 디자이너용 노코드 툴 | 학습 곡선 높음, 개발자 수준 요구 |

**한계:**
- 정해진 템플릿 내에서만 제작 가능
- 복잡한 기능(회원가입, 결제 등) 구현 어려움
- 진짜 "내 것"이 아닌 플랫폼 종속

**SSALWorks 차별점:**
- AI로 **완전 맞춤형** 웹사이트 개발
- 원하는 모든 기능 구현 가능
- 코드 소유권 100% 본인 것

---

### 2. 프로젝트 관리 도구

| 서비스 | 특징 | SSALWorks와의 차이 |
|--------|------|-------------------|
| **Notion** | 올인원 워크스페이스 | AI 연동 제한적, 개발 특화 아님 |
| **Trello** | 칸반 보드 기반 관리 | 단순 Task 관리, 3D 시각화 없음 |
| **Asana** | 기업용 프로젝트 관리 | 팀 협업 중심, 1인 개발에 과함 |
| **Jira** | 개발팀용 이슈 트래커 | 개발자 전용, 비개발자 진입장벽 높음 |

**한계:**
- 프로젝트 "관리"만 함
- 실제 "개발"과 연결 안 됨
- AI 코딩 도구와 통합 안 됨

**SSALWorks 차별점:**
- **PROJECT SSAL GRID**: 인접성/병렬성/의존성 3D 시각화
- Claude Code와 직접 연동
- 관리와 개발이 하나의 플랫폼에서

### 3. AI 코딩 도구

| 서비스 | 특징 | SSALWorks와의 차이 |
|--------|------|-------------------|
| **Claude Code** | Anthropic의 AI 코딩 어시스턴트 | CLI 기반, 가이드 없음 |
| **GitHub Copilot** | 코드 자동 완성 | 기존 개발자용, 비개발자 사용 어려움 |
| **Cursor** | AI 기반 코드 에디터 | 개발자용 IDE, 진입장벽 높음 |
| **Replit** | 온라인 코딩 환경 | 코딩 지식 필요 |

**한계:**
- "무엇을 만들어야 할지" 알려주지 않음
- 프로젝트 구조화 없음
- 비개발자가 사용하기 어려움

**SSALWorks 차별점:**
- Claude Code의 **운전대 + 네비게이션** 역할
- 단계별 가이드 제공
- Order Sheet로 체계적 지시 가능

### 4. 코딩 교육 서비스

| 서비스 | 특징 | SSALWorks와의 차이 |
|--------|------|-------------------|
| **코드잇** | 온라인 코딩 교육 | 교육만, 실제 프로젝트 없음 |
| **노마드코더** | 프로젝트 기반 교육 | 학습 후 스스로 해야 함 |
| **패스트캠퍼스** | 실무 중심 교육 | 고비용, 시간 소요 큼 |
| **인프런** | 강의 플랫폼 | 일방향 교육, 자문 없음 |

**한계:**
- 교육 "만" 제공
- 실제 프로젝트 수행은 본인 몫
- 막히면 물어볼 곳 없음

**SSALWorks 차별점:**
- **교육 + 도구 + 자문** 통합
- 배우면서 동시에 실제 프로젝트 수행
- 막히면 "써니에게 묻기"로 바로 해결

### 5. 창업 지원 서비스

| 서비스 | 특징 | SSALWorks와의 차이 |
|--------|------|-------------------|
| **창업진흥원** | 정부 창업 지원 | 자금 중심, 기술 지원 제한적 |
| **엑셀러레이터** | 투자 + 멘토링 | 선발된 팀만, 개발은 별도 |
| **스타트업 스튜디오** | 공동 창업 | 지분 공유, 의존성 높음 |

**한계:**
- 개발 "자체"를 해주지 않음
- 선발 과정 필요
- 비용 또는 지분 부담

**SSALWorks 차별점:**
- **누구나** 사용 가능
- 개발 도구 + 비즈니스 자문 통합
- 공인회계사 + 엑셀러레이터의 직접 자문

## 결론

### SSALWorks의 고유한 포지션

1. **직접적인 경쟁자 없음**
   - 도구 + 교육 + 샘플 + 자문을 결합한 서비스 부재

2. **블루오션 시장**
   - "비개발자의 AI 활용 웹개발" 시장은 아직 미개척

3. **선점 효과**
   - 먼저 시장을 선점하고 사용자 사례를 축적하면 후발 주자 진입 어려움

4. **지속 가능한 경쟁 우위**
   - 특허 + 전문성 + 커뮤니티로 장기적 우위 확보
