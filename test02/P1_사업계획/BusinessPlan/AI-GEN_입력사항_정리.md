# AI-GEN 사업계획서 입력사항 정리

> **작성일:** 2025-12-08
> **목적:** AI-GEN (www.ai-gen.co.kr) 사업계획서 생성을 위한 입력 내용 정리

---

## 1. 사업 아이디어 (필수, 최소 20자)

```
SSALWorks는 AI(Claude Code)를 활용하여 비개발자도 전문 웹사이트/웹서비스를 직접 개발할 수 있도록 지원하는 통합 플랫폼입니다.

3D 프로젝트 관리 시스템(PROJECT SSAL GRID), 단계별 학습 콘텐츠(Books), 다중 AI 통합(Claude/ChatGPT/Gemini/Perplexity), 그리고 공인회계사(CPA) + 엑셀러레이터의 비즈니스 멘토링을 결합하여 "도구 + 교육 + 자문"을 원스톱으로 제공합니다.

설치비 ₩3,000,000 + 월 플랫폼비 ₩50,000으로 시작하며, 3개월 내 성공 시 50% 환불 인센티브를 통해 고객의 성공을 보장합니다.

비전: "Everyone Can Be a Builder" - 누구나 자기 손으로 디지털 서비스를 만들 수 있는 세상
미션: "Muscle Memory Outsourcing (머슬 메모리의 외주화)" - 기술적 반복 작업은 AI에게, 인간은 창의적 의사결정에 집중
```

## 2. 경쟁사명 (선택)

### 직접 경쟁자
**없음** - SSALWorks와 동일한 방식(AI 코딩 도구 + 교육 + 비즈니스 자문)을 결합한 서비스는 존재하지 않음

### 간접 경쟁자 (부분 영역 중복)

**1. 웹 빌더 도구 (Web Builder)**
- Wix, Squarespace, Webflow
- 차이점: 노코드 템플릿 중심, 커스터마이징 한계, 학습 효과 없음

**2. 프로젝트 관리 도구 (Project Management)**
- Notion, Trello, Asana, Monday.com
- 차이점: 일반 업무 관리용, 개발 프로젝트 특화 없음, AI 통합 없음

**3. AI 코딩 도구 (AI Coding Tools)**
- Claude Code, Cursor, GitHub Copilot
- 차이점: 개발자 대상, 비개발자 진입장벽 높음, 교육/자문 없음

**4. 코딩 교육 플랫폼 (Coding Education)**
- 코드잇, 노마드코더, 프로그래머스, 인프런
- 차이점: 학습만 제공, 실제 프로젝트 관리/비즈니스 자문 없음

**5. 창업 지원 (Startup Support)**
- 액셀러레이터(Y Combinator, 500 Startups)
- 차이점: 비즈니스 자문만, 개발 도구/교육 없음, 진입장벽 높음

## 3. 차별성 (선택)

### 핵심 차별화 요소

**1. 유일무이한 통합 솔루션**
```
도구(PROJECT SSAL GRID) + 교육(Books) + 자문(CPA+엑셀러레이터)
→ 시장에서 유일하게 이 3가지를 모두 제공
```

**2. 비개발자 특화 설계**
- Claude Code를 "트랙터처럼 운전" 가능하도록 단순화
- 전문 용어 없이 이해 가능한 학습 콘텐츠
- 5060세대도 사용 가능한 진입장벽

**3. 3D 프로젝트 관리 시스템 (PROJECT SSAL GRID)**
- Stage(X축) × Area(Y축) × Level(Z축)
- 인접성/병렬성/의존성 시각화
- "뭘 해야 할지" 명확하게 제시

**4. CPA + 엑셀러레이터 자문**
```
"이건 Claude도, Anthropic도 절대 못 해주는 것입니다."
```
- 회계/재무 전문가(CPA) 관점의 사업 자문
- 실패 경험 기반의 실전 창업 멘토링
- 24-48시간 내 "써니에게 묻기" 응답

**5. 50% 환불 인센티브**
- 3개월 내 성공 조건 달성 시 ₩1,500,000 환불
- 조건: 웹사이트 완성 + 서비스 런칭 + 10명 이상 고객/수입
- 고객 성공에 진정성 있게 투자

**6. 성공 증거 (Proof of Concept)**
- SSALWorks 플랫폼 자체가 Claude Code로 제작
- "살아있는 교본" - 사용하면서 구조/기능 학습 가능

**7. 비용 효율**
| 비교 대상 | 비용 | SSALWorks |
|----------|------|-----------|
| 외주 개발 | ₩500만~₩2,000만 | ₩300만 (설치비) |
| 노코드 도구 | 월 ₩3만~₩10만 | 월 ₩50,000 |
| 코딩 부트캠프 | ₩500만~₩1,000만 | ₩300만 |
| AI 구독 (3개) | 월 ₩120,000 고정 | Claude만 구독 + 크레딧 방식 |

**8. 자립 가능 (Self-Sufficiency)**
- 수정/업데이트를 직접 수행 (외주비 ₩0)
- 개발 역량 = 평생 자산
- 플랫폼 종속 없음 (코드 소유)

## 5. 고객층 (선택)

### B2C (개인 고객)

**주요 타겟: "The Aspiring Founder" (예비 창업자)**

#### 세그먼트 1: 5060세대
- **특징:** 아이디어는 있지만 기술 장벽 높음
- **니즈:** "기술은 몰라도 내 사이트는 내가 만든다"
- **가치:** 컴퓨터로 문서 작성 가능하면 OK

#### 세그먼트 2: 자영업자 (소상공인)
- **특징:** 매장 운영 중, 디지털 전환 필요
- **니즈:** 예약/주문 시스템을 외주 없이 직접 구축
- **가치:** 외주 대비 80% 비용 절감, 수정 자유자재

#### 세그먼트 3: 프리랜서 / 청년 창업자
- **특징:** 아이디어는 있지만 개발자/개발비 부족
- **니즈:** MVP를 빠르게, 비용은 최소로
- **가치:** 투자자에게 보여줄 실제 동작 제품, 투자 유치 지원

### Anti-Persona (타겟 아님)
- **숙련된 개발자:** 이미 코딩 가능, SSALWorks 불필요
- **단순 정보 사이트 제작자:** Wix/노코드로 충분
- **대기업:** 자체 개발팀 보유
