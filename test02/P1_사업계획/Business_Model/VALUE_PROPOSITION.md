## 가치 제안 상세

### 가치 1: 플랫폼 사용권 (PROJECT SSAL GRID)

**문제:**
- 프로젝트 어디서부터 시작해야 할지 모름
- Task들이 어떻게 연결되는지 파악 어려움
- 진행 상황 추적이 안 됨

**해결:**
- **3D 그리드 시스템**: Stage(X) × Area(Y) × Level(Z)
- **인접성/병렬성/의존성** 시각적 파악
- 단계별 가이드로 **길을 잃지 않음**

**고객 혜택:**
- "뭘 해야 할지" 명확하게 알 수 있음
- 프로젝트 전체를 **한눈에** 파악
- 체계적으로 진행하여 **실패 확률 감소**

---

### 가치 2: 학습 콘텐츠 (Books)

**문제:**
- Claude Code 사용법을 모름
- 웹개발 용어와 개념이 낯섦
- 어디서 배워야 할지 모름

**해결:**
- **Claude & Claude Code 사용법** 완벽 가이드
- **웹개발 기초지식** 비개발자 눈높이
- **프로젝트 관리 방법론** 실전 적용
- **AI 프롬프팅 가이드** 효과적인 지시 방법

**고객 혜택:**
- **배우면서 만든다** - 학습과 실행 동시에
- 개발자 용어 이해하여 **AI와 소통 가능**
- 신규 콘텐츠 **지속 업데이트**

---

### 가치 3: 다중 AI 연동 (크레딧 방식)

**문제:**
- AI 서비스마다 개별 구독하면 **월 ₩12만원** 고정비
- 안 써도 매월 빠져나감
- 어떤 AI를 언제 써야 할지 모름

**해결:**
- **Claude만 구독** (핵심 개발 도구)
- ChatGPT, Gemini, Perplexity는 **크레딧 방식**
- 필요할 때만, **쓴 만큼만** 비용 발생

**고객 혜택:**
- **불필요한 고정비 제로**
- 상황에 맞는 AI 선택 가능
- 플랫폼 내에서 **원클릭** 연동

---

### 가치 4: 전문가 자문 (CPA + 엑셀러레이터)

**문제:**
- 웹사이트는 만들었는데 **사업은 어떻게?**
- 비즈니스 모델이 맞는지 확신 없음
- 회계/재무 관점 조언이 필요

**해결:**
- **"써니에게 묻기"** 기능
- 공인회계사(CPA)의 **회계/재무 자문**
- 엑셀러레이터의 **비즈니스 멘토링**
- 24-48시간 내 응답

**고객 혜택:**
- **기술 + 비즈니스** 둘 다 해결
- 별도 컨설팅 비용 **₩0**
- 실패 경험 기반의 **실전 조언**

> **"이건 Claude도, Anthropic도 절대 못 해주는 것입니다."**

---

### 가치 5: 성공 샘플 (SSALWorks 자체)

**문제:**
- "진짜 이걸로 만들 수 있어?" 의구심
- 성공 사례를 보고 싶음
- 어떤 결과물이 나오는지 궁금

**해결:**
- **SSALWorks 플랫폼 자체**가 Claude Code로 만든 것
- 사용하면서 **"이렇게 만들면 되는구나"** 체득
- 똑같은 방식으로 **자기 웹사이트** 제작 가능

**고객 혜택:**
- **살아있는 교본** - 사용하면서 배움
- "증거"를 직접 보고 **확신**을 가짐
- 구조와 기능을 **참고**하여 적용

---

### 가치 6: 50% 환불 기회

**문제:**
- 300만원이 아깝지 않을까?
- 실패하면 어떡하지?
- 동기부여가 필요함

**해결:**
- 3개월 내 **성공 조건** 달성 시 **50% 환불**
- 조건: 웹사이트 완성 + 서비스 런칭 + 10명 이상 고객 수입

**고객 혜택:**
- 성공하면 **실질 비용 ₩150만원**
- **강력한 동기부여** - 환불받으려면 성공해야 함
- **리스크 감소** - 실패해도 학습/경험 획득
