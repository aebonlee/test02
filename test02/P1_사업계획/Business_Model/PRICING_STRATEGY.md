## 가격 책정 원칙

### 1. 타겟 고객의 지불 능력 고려

**주 타겟:** 5060세대, 자영업자

**지불 능력 분석:**
- 퇴직금, 저축 등 자금 여력 있음
- 창업/사업에 투자할 의지 있음
- 가격보다 **가치**를 중시하는 세대

**심리적 가격대:**
| 금액 | 인식 |
|------|------|
| ₩100만원 | 너무 싸서 가치 의심 |
| ₩300만원 | **"한번 해볼 만하다"** |
| ₩500만원 | 진입장벽, 고민됨 |
| ₩1,000만원 | 부담, 외주와 비교 |

### 2. 가치 기반 가격 책정

**비교 대상: 외주 개발**

| 항목 | 외주 개발 | SSALWorks |
|------|----------|-----------|
| 초기 비용 | ₩500만~₩2,000만 | ₩300만원 |
| 수정 비용 | 건당 ₩50만~₩200만 | ₩0 (직접 수정) |
| 월 유지비 | ₩30만~₩100만 | ₩5만원 |
| 비즈니스 자문 | 별도 비용 | **포함** |
| 교육 | 없음 | **포함** |

**1년 총 비용 비교:**
- 외주 개발: ₩800만~₩3,000만원
- SSALWorks: ₩300만 + ₩60만 = **₩360만원**

### 3. 무료가 아닌 이유

**돈을 안 받으면 가치가 없다고 본다**

- 무료 서비스 = 진지하지 않은 사용자 유입
- 유료 = **진정성 있는 참여** 유도
- 300만원 = **"진짜 하겠다"는 의지의 증표**

## 설치비 ₩3,000,000 전략

### 왜 300만원인가?

| 요소 | 설명 |
|------|------|
| **진입 가능성** | 500만원은 부담, 300만원은 "해볼 만하다" |
| **가치 증명** | 너무 싸면 가치 의심, 적정 프리미엄 |
| **진정성 필터** | 진지하게 하려는 사람만 유입 |
| **수수료 절감** | 계좌이체로 카드 수수료 3.3% 절약 (약 10만원) |

### 50% 환불 인센티브

**조건:** 3개월 내 아래 모두 달성 시
1. ✅ 웹사이트 완성 (공개 URL 배포)
2. ✅ 서비스 런칭 (도메인 연결)
3. ✅ 10명 이상 고객 수입 발생

**환불액:** ₩1,500,000

**철학:**
> "3개월 안에 반드시 성공하게 만들겠다"

**효과:**
- 사용자: 성공해야 환불 → **진지하게 임함**
- 운영자: 성공시켜야 함 → **최선을 다함**
- **상호 몰입 구조** 형성

## 플랫폼 이용료 ₩50,000/월 전략

### 왜 5만원인가?

| 요소 | 설명 |
|------|------|
| **AI 서비스 대비** | Claude Pro ₩3만원, 플랫폼+자문 포함하면 저렴 |
| **외주 유지비 대비** | 월 ₩30만~₩100만 대비 80% 저렴 |
| **포함 가치** | 플랫폼 + 학습 + AI 연동 + **CPA 자문** |

### 5만원에 포함된 것

| 서비스 | 별도 이용 시 비용 | SSALWorks |
|--------|-----------------|-----------|
| 프로젝트 관리 도구 | ₩1만~₩3만/월 | ✅ 포함 |
| 학습 콘텐츠 | ₩5만~₩10만/월 | ✅ 포함 |
| AI 연동 (ChatGPT, Gemini 등) | 각 ₩2만~₩3만/월 | ✅ 크레딧 방식 |
| 비즈니스 자문 | ₩50만~₩100만/건 | ✅ **무제한 포함** |

**월 5만원 = 실제 가치 ₩30만원 이상**

### 3개월 면제 전략

**목적:**
- 3개월간 적응 및 학습 기간
- 부담 없이 시작하게 함
- 4개월차부터 가치를 체감하고 결제

**타임라인:**
```
설치비 납부 (₩300만원) + 웰컴 크레딧 ₩50,000 자동 지급
    ↓
1~3개월차: 면제 (₩0) - 적응/학습/성장
    ↓
4개월차: ₩5만원 - 첫 과금
    ↓
5개월차~: ₩5만원/월 계속
```

| 항목 | 내용 |

- 불필요한 고정비 **제로**

