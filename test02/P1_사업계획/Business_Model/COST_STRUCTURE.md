## 2. 변동비 상세

### AI API 크레딧 비용

사용자에게 제공하는 AI 크레딧의 실제 비용

#### 비용 계산 방식

```
사용자 청구액 = API 실비용 × 1.3 (마진 30%)

예시:
- 사용자 충전: ₩10,000
- 실제 API 비용: ₩10,000 ÷ 1.3 = ₩7,692
- SSALWorks 마진: ₩2,308 (30%)
```

#### 웰컴 크레딧 제공 비용

| 항목 | 내용 |
|------|------|
| 신규 사용자당 제공 | ₩50,000 (웰컴 크레딧) |
| 실제 API 비용 | ₩41,667 (÷1.2) |
| 평균 사용률 | 70% |
| **실제 비용** | **₩29,167/명** |

#### 사용자 100명 기준 예상

```
웰컴 크레딧 비용:
100명 × ₩29,167 = ₩2,916,700

추가 충전 마진:
충전자 20% (20명) × 평균 ₩20,000/월 = ₩400,000 충전
마진 30% = ₩120,000/월 수익
```

## 5. 손익분기점 분석

### 월 고정비 기준

| 항목 | 초기 | 성장기 |
|------|------|--------|
| 인프라 | ₩102,000 | ₩362,000 |
| 툴 이용료 | ₩80,000 | ₩110,000 |
| 마케팅 | ₩0 | ₩1,000,000 |
| **합계** | **₩182,000** | **₩1,472,000** |

### 손익분기점 계산

#### 초기 단계 (고정비 ₩182,000)

```
플랫폼 이용료: ₩50,000/월 (3개월 면제 후, 4개월차부터)

손익분기점 = 고정비 ÷ 이용료
          = ₩182,000 ÷ ₩50,000
          = 3.6명

→ 월 4명 이상 유지하면 손익분기
```

#### 성장기 (고정비 ₩1,472,000)

```
손익분기점 = ₩1,472,000 ÷ ₩50,000
          = 29.4명

→ 월 30명 이상 유지하면 손익분기
```

### 설치비 포함 손익분기

```
설치비 1명 = ₩3,000,000 (환불 미발생 시)
         = ₩1,500,000 (환불 발생 시, 확률 30%)

기대 설치비 = ₩3,000,000 × 0.7 + ₩1,500,000 × 0.3
           = ₩2,100,000 + ₩450,000
           = ₩2,550,000/명

월 1명만 가입해도:
₩2,550,000 > 월 고정비 ₩1,472,000
→ 흑자
```

## 7. 수익성 분석

```
플랫폼 이용료: 70명 × ₩50,000 = ₩3,500,000
(30명은 1~3개월차 무료 이용 기간 가정)

총 수익: ₩3,580,000/월
```

**비용:**
```
고정비: ₩472,000 (인프라 + 툴)

마케팅: ₩1,000,000

총 비용: ₩1,772,000/월
```

**월 순이익:**
```
₩3,580,000 - ₩1,772,000 = ₩1,808,000/월
```

```
연간 순이익: ₩1,808,000 × 12 = ₩21,696,000
```

## 결론

### 린(Lean) 운영의 강점

1. **낮은 고정비** → 빠른 손익분기
2. **1인 운영** → 인건비 제로
3. **클라우드 기반** → 유연한 확장
4. **마진 구조** → AI 크레딧에서 추가 수익

### 핵심 메시지

> **"최소 비용으로 최대 가치를 전달하는 구조"**

SSALWorks는 린(Lean) 운영으로 낮은 고정비를 유지하면서, 설치비와 월 이용료로 안정적인 수익을 창출합니다.
