# 클로드 코드 삭제 및 개발 환경 초기화 방법

> 다른 컴퓨터에서 새로 시작하거나, 문제가 생겨서 처음부터 다시 설치해야 할 때가 있습니다. 이 문서는 Claude Code와 개발 환경을 완전히 초기화하는 방법을 안내합니다.

---

## 왜 초기화가 필요한가

개발 환경을 초기화해야 하는 상황은 여러 가지가 있습니다. 설치 과정에서 오류가 발생했거나, 버전 충돌 문제가 있거나, 단순히 깨끗한 상태에서 다시 시작하고 싶을 때입니다.

특히 여러 컴퓨터에서 작업하는 경우, 각 컴퓨터의 환경을 동일하게 맞추기 위해 초기화가 필요할 수 있습니다. 이전 설치의 잔여 파일이 새 설치에 영향을 줄 수 있기 때문입니다.

---

## 삭제해야 할 프로그램 3가지

개발 환경을 완전히 초기화하려면 다음 세 가지를 모두 삭제해야 합니다.

첫째, Claude Code입니다. npm을 통해 설치한 CLI 도구입니다.

둘째, Node.js입니다. Claude Code의 실행 환경이며, npm도 함께 삭제됩니다.

셋째, Git입니다. 버전 관리 도구로, SSALWorks Dev Package 사용 시 필수입니다.

세 가지를 모두 삭제해야 완전한 초기화가 됩니다. 하나라도 남아있으면 새 설치 시 충돌이 발생할 수 있습니다.

---

## 삭제 방법 1: Windows 설정에서 삭제

가장 확실한 방법은 Windows 설정에서 직접 삭제하는 것입니다.

1. Windows 키를 누르고 "앱"을 검색합니다.
2. "앱 및 기능" 또는 "설치된 앱"을 클릭합니다.
3. 검색창에서 다음을 검색하여 삭제합니다:
   - "Node.js" 검색 → 제거 클릭
   - "Git" 검색 → 제거 클릭

Claude Code는 별도 프로그램이 아니라 npm 패키지이므로 Node.js를 삭제하면 함께 삭제됩니다.

---

## 삭제 방법 2: 명령어로 삭제

터미널에서 명령어로 삭제할 수도 있습니다. 관리자 권한으로 명령 프롬프트나 PowerShell을 실행해야 합니다.

Claude Code만 삭제하려면:
```
npm uninstall -g @anthropic-ai/claude-code
```

Node.js와 Git은 설치 방법에 따라 삭제 명령이 다릅니다. winget으로 설치했다면:
```
winget uninstall OpenJS.NodeJS
winget uninstall Git.Git
```

하지만 공식 사이트에서 직접 다운로드해서 설치했다면 winget 명령이 작동하지 않습니다. 이 경우 Windows 설정에서 직접 삭제해야 합니다.

---

## 삭제 확인 방법

모든 프로그램이 정상적으로 삭제되었는지 확인하려면 터미널에서 다음 명령을 실행합니다.

```
node --version
npm --version
git --version
claude --version
```

모든 명령에서 "인식할 수 없는 명령입니다" 또는 "is not recognized"라고 나오면 삭제가 완료된 것입니다.

만약 버전 번호가 표시되면 아직 삭제가 완료되지 않은 것입니다. 터미널을 새로 열고 다시 확인해보세요. 그래도 나오면 Windows 설정에서 다시 확인합니다.

---

## 잔여 폴더 정리

프로그램을 삭제해도 일부 설정 파일이나 캐시가 남아있을 수 있습니다. 완전한 초기화를 원한다면 다음 폴더들도 삭제합니다.

Node.js 관련:
- `C:\Users\사용자명\AppData\Roaming\npm`
- `C:\Users\사용자명\AppData\Roaming\npm-cache`

Claude Code 관련:
- `C:\Users\사용자명\.claude`

Git 관련:
- `C:\Users\사용자명\.gitconfig` 파일

이 폴더들은 삭제하지 않아도 새 설치에 큰 영향은 없습니다. 하지만 이전 설정이 남아있으면 예상치 못한 동작이 발생할 수 있으므로, 완전한 초기화를 원한다면 삭제를 권장합니다.

---

## 다시 설치하기

초기화가 완료되면 다음 순서로 다시 설치합니다.

1. Git 설치: git-scm.com에서 다운로드
2. Node.js 설치: nodejs.org에서 LTS 버전 다운로드
3. Claude Code 설치: `npm install -g @anthropic-ai/claude-code`

설치 후 각 프로그램의 버전을 확인하여 정상 설치를 확인합니다.

```
git --version
node --version
npm --version
claude --version
```

---

## 주의사항

초기화 전에 중요한 프로젝트 파일이 있다면 반드시 백업하세요. 프로그램을 삭제해도 프로젝트 폴더의 코드 파일은 삭제되지 않지만, `.claude` 폴더의 세션 기록은 삭제됩니다.

또한 Git으로 관리하던 프로젝트가 있다면 GitHub에 푸시해두는 것이 안전합니다. 로컬 Git 설정이 삭제되어도 원격 저장소에는 영향이 없습니다.
