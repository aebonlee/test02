# 프로젝트 폴더에서 터미널 열고 Claude Code 실행하기

> 이 문서는 프로젝트 폴더에서 터미널을 열고 Claude Code를 실행하는 방법을 설명합니다.

---

## 왜 프로젝트 폴더에서 실행하는가

Claude Code는 현재 폴더를 작업 디렉토리로 인식합니다. 프로젝트 폴더에서 실행해야 올바른 파일들을 읽고 수정할 수 있습니다.

---

## Windows에서 터미널 열기

**방법 1: 파일 탐색기에서**
1. 프로젝트 폴더로 이동
2. 주소창 클릭 → `cmd` 또는 `powershell` 입력 → Enter

**방법 2: 우클릭 메뉴**
1. 프로젝트 폴더에서 Shift + 우클릭
2. "여기에 PowerShell 창 열기" 선택

**방법 3: VS Code에서**
1. VS Code로 프로젝트 폴더 열기
2. Ctrl + ` (백틱)으로 터미널 열기

---

## Claude Code 실행

터미널에서:
```bash
claude
```

---

## 경로 확인

실행 후 현재 경로 확인:
```
"현재 작업 디렉토리가 어디야?"
```

Claude Code가 올바른 프로젝트 폴더에 있는지 확인합니다.

---

## 흔한 실수

| 실수 | 해결 방법 |
|------|----------|
| 홈 디렉토리에서 실행 | 프로젝트 폴더로 이동 후 실행 |
| 다른 프로젝트 폴더에서 실행 | 올바른 폴더로 이동 |
| 경로에 한글 포함 | 영문 경로로 프로젝트 이동 권장 |

---

## VS Code 통합 터미널 권장

VS Code 터미널을 사용하면:
- 항상 프로젝트 폴더에서 시작
- 파일 변경 즉시 확인 가능
- 코드 복사/붙여넣기 편리

---

## 폴더 이동 명령어

```bash
# Windows
cd C:\Users\사용자명\프로젝트폴더

# 드라이브 변경
D:
cd D:\프로젝트폴더
```

---

## 실행 전 확인

| 확인 항목 | 방법 |
|----------|------|
| 현재 폴더 | 터미널에서 `pwd` 또는 `cd` |
| 프로젝트 파일 존재 | `ls` 또는 `dir` |
| CLAUDE.md 존재 | 프로젝트 설정 파일 확인 |
