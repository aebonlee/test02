# Windows 기본 음성 입력 기능으로 Claude Code에 음성으로 지시하기

> 이 문서는 Windows 기본 음성 인식을 활용하여 Claude Code에 지시하는 방법을 설명합니다.

---

## 왜 음성 입력을 사용하는가

긴 지시사항을 타이핑하는 것보다 말로 하는 것이 빠릅니다. 특히 아이디어를 설명하거나 요구사항을 전달할 때 유용합니다.

---

## Windows 음성 입력 사용법

**단축키: Win + H**

1. Claude Code 터미널에서 입력 커서가 있는 상태로 대기
2. Win + H 눌러 음성 입력 활성화
3. 지시사항을 말하기
4. 다시 Win + H 눌러 종료
5. Enter로 전송

---

## 음성 입력 팁

| 팁 | 설명 |
|-----|------|
| 천천히 말하기 | 급하게 말하면 인식률 떨어짐 |
| 마침표 말하기 | "마침표"라고 말하면 . 입력됨 |
| 엔터 말하기 | "새 줄"이라고 말하면 Enter 입력됨 |
| 영어는 영어로 | API, function 등은 영어 발음으로 |

---

## 특수 문자 입력

| 말하기 | 입력 결과 |
|--------|----------|
| "마침표" | . |
| "쉼표" | , |
| "물음표" | ? |
| "새 줄" | Enter |
| "탭" | Tab |

---

## 음성 입력 예시

```
말하기:
"로그인 기능을 만들어줘 마침표 이메일과 비밀번호 입력 폼이 있고
쉼표 구글 로그인 버튼도 추가해줘 마침표"

결과:
"로그인 기능을 만들어줘. 이메일과 비밀번호 입력 폼이 있고, 구글 로그인 버튼도 추가해줘."
```

---

## 언제 사용하면 좋은가

| 상황 | 추천 |
|------|------|
| 긴 요구사항 설명 | 음성 입력 |
| 코드 수정 지시 | 타이핑 |
| 간단한 명령 | 타이핑 |
| 아이디어 브레인스토밍 | 음성 입력 |

---

## 주의사항

- 조용한 환경에서 사용
- 마이크 설정 확인 (설정 → 개인 정보 → 마이크)
- 영어 혼용 시 인식률 떨어질 수 있음

---

## 음성 인식이 안 될 때

1. 마이크 연결 확인
2. Windows 설정 → 개인 정보 → 마이크 권한 확인
3. 조용한 환경으로 이동
4. 더 정확한 인식이 필요하면 AI 서비스 활용
