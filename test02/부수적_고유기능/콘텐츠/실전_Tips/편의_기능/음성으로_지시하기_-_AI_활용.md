# Whisper, Vrew 같은 AI 음성인식으로 정확하게 음성 입력하기

> 이 문서는 AI 서비스의 음성 인식을 활용하여 Claude Code에 지시하는 방법을 설명합니다.

---

## 왜 AI 서비스를 활용하는가

영어와 한글이 섞인 기술 용어는 Windows 기본 음성 인식보다 AI 서비스가 더 정확합니다.

```
말한 것:
"API 만들어줘 파라미터는 userId랑 email이고 리턴은 boolean으로"

Windows 인식: "에이피아이 만들어줘 파라미터는 유저아이디랑..."
ChatGPT 인식: "API 만들어줘. 파라미터는 userId와 email이고, 리턴은 boolean으로"
```

---

## 사용법

1. chat.openai.com 또는 gemini.google.com 접속
2. 마이크 아이콘 클릭
3. 음성으로 지시사항 말하기
4. 인식된 텍스트 복사 (Ctrl+C)
5. Claude Code에 붙여넣기 (Ctrl+V)

---

## 서비스 비교

| 특징 | ChatGPT | Gemini | Whisper |
|------|---------|--------|---------|
| 영어 혼용 | 매우 좋음 | 좋음 | 매우 좋음 |
| 코드 용어 | 매우 좋음 | 좋음 | 매우 좋음 |
| 한국어 | 좋음 | 매우 좋음 | 좋음 |
| 무료 | O | O | O (API) |

---

## 추천 상황

| 상황 | 추천 |
|------|------|
| 짧은 한국어 지시 | Windows 기본 (Win+H) |
| 영어 혼용, 코드 용어 | ChatGPT 음성 |
| 긴 설명 | AI 서비스 음성 |
| 오프라인 | Whisper 로컬 설치 |

---

## Vrew 활용법

Vrew는 영상 편집 프로그램이지만 음성 인식 기능이 뛰어납니다.

1. Vrew 설치 (무료)
2. 새 프로젝트 → 음성 녹음
3. 말하기 완료 후 자막 생성
4. 생성된 텍스트 복사

---

## 음성 인식이 잘 안 될 때

| 문제 | 해결 방법 |
|------|----------|
| 영어가 한글로 인식 | 영어 부분만 따로 말하기 |
| 문장이 끊김 | 더 길게 이어서 말하기 |
| 오타가 많음 | 천천히 또박또박 말하기 |

---

## 팁

- 천천히 또박또박 말하기
- 문장 단위로 끊어 말하기
- 영어 단어는 영어 발음으로
