## 핵심 정리

| 구분 | 내용 |
|------|------|
| AI가 잘하는 것 | 시키는 것 (코드 작성 80-95% 자동화) |
| 사람이 해야 하는 것 | 무엇을 시킬지 결정하는 것 |

```
Happy Path → AI가 만든다
Edge Case → 사람이 요청한다
좋은 UX → 사람이 판단한다
```
