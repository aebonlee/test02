# 브라우저 개발자 도구(F12) 콘솔에서 JavaScript 에러와 네트워크 오류 확인하기

> 이 문서는 브라우저 개발자 도구를 사용하여 JavaScript 에러와 네트워크 오류를 확인하는 방법을 설명합니다.

---

## 왜 콘솔 확인이 중요한가

화면에 아무것도 안 뜨거나 버튼이 작동하지 않을 때, 콘솔에 에러 메시지가 있습니다. 정확한 에러 메시지를 Claude Code에게 전달하면 문제를 빠르게 해결할 수 있습니다.

---

## 개발자 도구 열기

| 브라우저 | 단축키 |
|----------|--------|
| Chrome/Edge | F12 또는 Ctrl+Shift+J |
| Firefox | F12 또는 Ctrl+Shift+K |
| Safari | Cmd+Option+I |
| Mac 공통 | Cmd+Option+J |

---

## 탭 종류와 용도

| 탭 | 용도 | 언제 보나 |
|----|------|----------|
| **Console** | JavaScript 에러, 로그 | 기능 오류 시 |
| **Network** | API 요청/응답 | 데이터 안 나올 때 |
| **Elements** | HTML/CSS 구조 | 화면 레이아웃 문제 |
| **Sources** | 코드 디버깅 | 중단점 설정 시 |

---

## Console 탭 사용법

1. F12로 개발자 도구 열기
2. 상단 **Console** 탭 클릭
3. 빨간색 에러 메시지 확인
4. 에러 클릭하면 발생 위치(파일:줄번호) 표시

---

## Network 탭 사용법

1. F12로 개발자 도구 열기
2. 상단 **Network** 탭 클릭
3. 페이지 새로고침 (F5)
4. 빨간색으로 표시된 요청 클릭
5. **Response** 탭에서 에러 메시지 확인

---

## 메시지 유형

| 색상 | 의미 | 조치 |
|------|------|------|
| 빨간색 | Error - 심각한 오류 | 즉시 확인 필요 |
| 노란색 | Warning - 경고 | 무시해도 되는 경우 많음 |
| 파란색 | Info - 정보 | 디버깅용 로그 |

---

## 자주 보는 에러

| 에러 | 원인 | 해결 |
|------|------|------|
| TypeError | 존재하지 않는 속성 접근 | null 체크 추가 |
| ReferenceError | 정의되지 않은 변수 사용 | 변수명 확인 |
| 404 Not Found | 잘못된 파일 경로 | 경로 확인 |
| CORS 오류 | 다른 도메인 API 호출 | 서버 설정 확인 |

---
