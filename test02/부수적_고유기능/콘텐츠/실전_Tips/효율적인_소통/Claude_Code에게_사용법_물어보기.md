# Claude Code에게 사용법이나 기능 직접 물어보기

> 이 문서는 Claude Code 자체에게 기능이나 사용법을 물어보는 방법을 설명합니다.

---

## Claude Code에게 직접 물어보기

Claude Code는 자신의 기능에 대해 잘 알고 있습니다. 궁금한 점이 있으면 직접 물어보세요.

---

## 물어볼 수 있는 것들

| 질문 유형 | 예시 |
|----------|------|
| 기능 | "서브에이전트가 뭐야?" |
| 사용법 | "git 커밋 어떻게 해?" |
| 설정 | "MCP 서버 어떻게 추가해?" |
| 제한 | "동시에 몇 개 작업 가능해?" |

---

## 효과적인 질문 방법

```
"이거 어떻게 해?" (X - 너무 모호)
"Supabase에 데이터 어떻게 저장해?" (O - 구체적)
```

---

## 질문 예시

```
"Claude Code에서 파일 검색 어떻게 해?"
"Slash Command 만드는 방법 알려줘"
"Skills가 뭐고 어떻게 써?"
"서브에이전트 몇 개까지 동시에 돌릴 수 있어?"
```

---

## 답변을 이해 못했을 때

```
"좀 더 쉽게 설명해줘"
"예시 코드로 보여줘"
"단계별로 알려줘"
```

---

## 모르는 기능 발견

```
"방금 뭘 한 거야?"
"그게 무슨 기능이야?"
"왜 그렇게 했어?"
```

---

## 공식 문서 요청

```
"이 기능에 대한 공식 문서 있어?"
"더 자세한 정보는 어디서 봐?"
```

---

## 질문 후 확인

답변을 받으면:
1. 이해했는지 확인
2. 직접 시도해보기
3. 안 되면 다시 질문
