# 이메일 인증 코드는 문자 대신 숫자 6자리로 요청하기

> 이 문서는 사용자 친화적인 인증 코드 형식을 요청하는 방법을 설명합니다.

---

## 왜 숫자 6자리인가

| 형식 | 문제점 |
|------|--------|
| 영문+숫자 혼합 | 입력 어려움 (0과 O, 1과 l 혼동) |
| 긴 코드 | 기억/입력 어려움 |
| 특수문자 포함 | 모바일 키보드 전환 필요 |

숫자 6자리는:
- 모바일에서 숫자 키패드로 쉽게 입력
- 충분한 보안성 (100만 가지 조합)
- 대부분의 서비스 표준

---

## Claude Code에게 요청

```
"인증 코드는 숫자 6자리로 만들어줘"
"영문 대신 숫자만 사용해줘"
```

---

## 코드 예시

```javascript
// 좋은 예시
function generateCode() {
  return Math.floor(100000 + Math.random() * 900000).toString();
}
// 결과: "384729"

// 나쁜 예시
function generateCode() {
  return Math.random().toString(36).substring(2, 8).toUpperCase();
}
// 결과: "A3B8C2" - 입력 어려움
```
