# 5편 | 모바일 앱과 웹

---

Claude를 사용하는 가장 간단한 방법은 웹 브라우저나 모바일 앱을 이용하는 것이다. 별도의 설치나 설정 없이 계정만 만들면 즉시 사용할 수 있다. 이 편에서는 웹과 모바일 앱을 통해 Claude를 효과적으로 활용하는 방법을 다룬다.

## 1. 웹 버전 (claude.ai)

### 1-1 시작하기

웹 브라우저에서 claude.ai에 접속하면 Claude를 사용할 수 있다. Chrome, Safari, Edge, Firefox 등 대부분의 현대적인 브라우저를 지원한다.

**계정 생성**: Google 계정이나 이메일로 가입할 수 있다. 무료 계정으로도 기본적인 기능을 모두 사용할 수 있으며, Pro 이상의 유료 플랜을 구독하면 더 많은 사용량과 추가 기능을 이용할 수 있다.

**인터페이스 구성**: 화면 왼쪽에는 대화 목록이, 중앙에는 현재 대화 내용이, 하단에는 메시지 입력창이 위치한다. 직관적인 구성으로 처음 사용하는 사람도 쉽게 적응할 수 있다.

### 1-2 기본 대화 기능

Claude와의 대화는 메시지를 입력하고 보내는 것으로 시작한다. 질문을 하면 답변을 받고, 추가 질문을 이어갈 수 있다.

**대화의 맥락**: Claude는 같은 대화 내에서 이전 내용을 기억한다. "방금 말한 내용을 더 자세히 설명해줘"라고 하면 이전 답변을 참고하여 보충 설명을 제공한다.

**새 대화 시작**: 다른 주제로 넘어가고 싶다면 새 대화를 시작하는 것이 좋다. 왼쪽 상단의 "New chat" 버튼을 클릭하면 된다. 이전 대화는 목록에 저장되어 나중에 다시 확인할 수 있다.

### 1-3 파일 업로드

Claude는 다양한 형식의 파일을 분석할 수 있다. 메시지 입력창 옆의 클립 아이콘을 클릭하거나 파일을 드래그앤드롭하면 된다.

**지원 파일 형식**:
- 문서: PDF, Word, TXT, Markdown
- 이미지: JPG, PNG, GIF, WebP
- 데이터: CSV, JSON, XML
- 코드: 대부분의 프로그래밍 언어 파일

**파일 크기 제한**: 개별 파일은 최대 10MB까지 업로드 가능하다. 여러 파일을 동시에 업로드할 수도 있다.

**활용 예시**:
- PDF 보고서를 업로드하고 "이 문서의 핵심 내용을 요약해줘"
- 이미지를 업로드하고 "이 차트가 무엇을 보여주는지 설명해줘"
- 코드 파일을 업로드하고 "이 코드에 버그가 있는지 확인해줘"

### 1-4 Artifacts 기능

Artifacts는 Claude가 코드, 문서, 차트 등을 별도의 패널에서 생성하고 표시하는 기능이다.

**작동 방식**: "간단한 할일 목록 웹앱을 만들어줘"라고 요청하면, Claude는 코드를 대화창에 텍스트로 보여주는 대신 오른쪽 Artifacts 패널에 실제로 작동하는 형태로 표시한다.

**Artifacts 유형**:
- 코드: HTML, CSS, JavaScript 등이 실행 가능한 형태로 표시
- 문서: Markdown 문서가 렌더링된 형태로 표시
- 차트: 데이터 시각화가 그래프로 표시
- 다이어그램: 플로우차트, 시퀀스 다이어그램 등

**편집 기능**: Artifacts 패널에서 직접 코드를 수정할 수 있다. 수정 후 "이 변경사항을 반영해서 다시 설명해줘"라고 하면 Claude가 수정된 내용을 이해하고 답변한다.

### 1-5 Projects 기능

관련된 대화들을 하나의 프로젝트로 묶어서 관리할 수 있다.

**프로젝트 생성**: 왼쪽 사이드바에서 "New project"를 클릭하여 프로젝트를 만든다. 프로젝트에 이름을 붙이고, 관련 파일이나 지시사항을 추가할 수 있다.

**프로젝트 지시사항**: 프로젝트에 기본 지시사항을 설정하면, 해당 프로젝트 내 모든 대화에서 그 지시사항이 적용된다. 예를 들어 "모든 답변을 한국어로 해주고, 기술 용어는 영어 원문도 병기해줘"라고 설정할 수 있다.

**지식 베이스**: 프로젝트에 파일을 추가하면 해당 프로젝트의 모든 대화에서 그 파일들을 참조할 수 있다. 회사 가이드라인 문서, 제품 사양서 등을 추가해두면 매번 업로드할 필요 없이 활용할 수 있다.

## 2. 모바일 앱

### 2-1 앱 설치 및 설정

Claude 모바일 앱은 iOS App Store와 Google Play Store에서 무료로 다운로드할 수 있다. 설치 후 웹에서 사용하던 계정으로 로그인하면 대화 기록이 동기화된다.

**지원 기기**: iPhone, iPad, Android 스마트폰 및 태블릿을 지원한다.

### 2-2 모바일 특화 기능

**음성 입력**: 키보드 대신 음성으로 질문할 수 있다. 입력창의 마이크 아이콘을 탭하고 말하면 된다. 한국어 음성 인식도 지원한다.

**카메라 연동**: 앱에서 직접 사진을 촬영하여 Claude에게 분석을 요청할 수 있다. 문서 촬영, 제품 사진 분석, 손글씨 인식 등에 활용할 수 있다.

**공유 기능**: 다른 앱에서 텍스트나 이미지를 선택하고 "공유" → "Claude"를 선택하면 바로 Claude에게 전달된다. 웹 기사를 공유하고 요약을 요청하거나, 이메일 내용을 공유하고 답장 초안을 요청하는 식으로 활용할 수 있다.

### 2-3 이동 중 활용 시나리오

**회의 준비**: 이동 중에 회의 자료를 Claude에게 보여주고 핵심 포인트를 정리받는다.

**이메일 작성**: 음성으로 이메일 내용을 설명하고 Claude가 정돈된 이메일 초안을 작성하게 한다.

**빠른 조사**: 궁금한 내용을 음성으로 질문하고 즉시 답변을 받는다.

**번역 지원**: 외국어 문서나 간판을 촬영하여 번역을 요청한다.

## 3. 웹과 모바일의 동기화

### 3-1 계정 동기화

동일한 계정으로 로그인하면 웹과 모바일 간에 다음 내용이 동기화된다:

- 모든 대화 기록
- 프로젝트 및 프로젝트 내 파일
- 계정 설정
- 구독 상태

한쪽에서 시작한 대화를 다른 쪽에서 이어갈 수 있다. 예를 들어, 사무실 PC에서 작업하다가 외출 시 모바일에서 같은 대화를 계속할 수 있다.

### 3-2 기기별 최적 활용

**PC/웹에서 좋은 작업**:
- 긴 문서 작성 및 편집
- 복잡한 코드 작업
- 여러 파일 동시 분석
- Artifacts를 활용한 시각적 작업

**모바일에서 좋은 작업**:
- 빠른 질문과 답변
- 음성 기반 대화
- 사진 촬영 및 분석
- 이동 중 간단한 작업

## 4. 효과적인 사용 팁

### 4-1 대화 관리

**대화 제목**: 나중에 찾기 쉽도록 대화에 의미 있는 제목을 붙여두자. 대화 목록에서 대화를 우클릭(모바일에서는 길게 탭)하면 이름을 변경할 수 있다.

**대화 정리**: 오래된 대화나 불필요한 대화는 삭제하여 목록을 깔끔하게 유지한다.

**검색 활용**: 대화 목록 상단의 검색 기능을 활용하면 과거 대화를 쉽게 찾을 수 있다.

### 4-2 응답 품질 향상

**명확한 질문**: 구체적으로 질문할수록 좋은 답변을 받을 수 있다. "마케팅에 대해 알려줘"보다 "B2B SaaS 제품의 콘텐츠 마케팅 전략 3가지를 설명해줘"가 더 유용한 답변을 얻는다.

**맥락 제공**: 질문의 배경을 설명하면 더 적절한 답변을 받을 수 있다. "나는 스타트업 마케터이고, 예산이 적은 상황이야"라는 맥락을 추가하면 그에 맞는 조언을 받을 수 있다.

**피드백 제공**: 답변이 마음에 들지 않으면 "더 간단하게 설명해줘" 또는 "실제 예시를 들어줘"라고 요청한다. Claude는 피드백을 반영하여 답변을 개선한다.

### 4-3 일상 업무 통합

**루틴화**: 매일 반복하는 작업에 Claude를 통합한다. 아침에 뉴스레터 초안 작성, 회의 후 회의록 정리, 퇴근 전 다음 날 할일 정리 등에 활용할 수 있다.

**템플릿 활용**: 자주 사용하는 요청은 템플릿으로 저장해두고 재사용한다. 프로젝트의 지시사항 기능을 활용하면 매번 같은 설명을 반복할 필요가 없다.
