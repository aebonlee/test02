# 19편 | Skills

---

Claude Code는 특정 파일 유형을 처리하는 전문 기능을 제공한다. PDF를 분석하거나 엑셀 파일을 처리할 때, Claude Code는 자동으로 적합한 Skill을 활성화한다. 이 편에서는 Skills의 개념과 활용법을 살펴본다.

## 1. Skills 개요

### 1-1 Skills란

Skills는 특정 작업에 특화된 전문 기능이다. 일반적인 대화로는 처리하기 어려운 작업을 전문적으로 수행한다.

```
일반 대화: "이 PDF 읽어줘"
           → Claude Code가 pdf Skill 자동 활성화
           → PDF 전문 처리 로직 실행
           → 결과 반환
```

### 1-2 Skills vs Commands 차이

| 구분 | Commands | Skills |
|------|----------|--------|
| 호출 방식 | 슬래시(/)로 직접 호출 | 자동 감지 또는 Skill 도구 |
| 실행 시점 | 사용자가 명시적으로 실행 | 상황에 따라 자동 실행 |
| 목적 | 특정 동작 즉시 실행 | 특정 파일/작업 전문 처리 |
| 예시 | /clear, /model | pdf, xlsx |

**Commands:** 사용자가 직접 "/명령어"로 호출
**Skills:** Claude Code가 상황을 판단해 자동 활성화

### 1-3 Skills 작동 방식

```
[파일 작업 요청]
      ↓
[Claude Code 분석]
      ↓
[적합한 Skill 있음?]
   ├── Yes → Skill 자동 활성화 → 전문 처리
   └── No  → 일반 처리
```

## 2. Skills 활성화

### 2-1 자동 감지

Claude Code는 작업 내용을 분석해서 적합한 Skill을 자동으로 활성화한다.

```
사용자: 이 PDF 파일 요약해줘
Claude: [pdf Skill 자동 활성화]
        PDF 내용을 분석합니다...

        요약:
        - 주요 내용 1
        - 주요 내용 2
        ...
```

자동 감지되는 상황:
- PDF 파일 경로가 포함된 요청
- 엑셀/스프레드시트 관련 요청
- 오피스 문서 처리 요청

### 2-2 수동 호출

Skill 도구를 사용해서 직접 호출할 수도 있다.

```
사용자: pdf skill을 사용해서 이 문서 분석해줘
Claude: [Skill 도구로 pdf 호출]
        PDF 분석을 시작합니다...
```

### 2-3 활성화 조건

Skills는 다음 조건에서 활성화된다:

**자동 활성화:**
- 해당 파일 유형이 요청에 포함될 때
- 파일 경로가 명시되었을 때

**수동 활성화:**
- 사용자가 특정 Skill 사용을 요청할 때
- Skill 도구를 직접 호출할 때

## 3. 기본 제공 Skills

### 3-1 pdf (PDF 처리)

PDF 파일을 읽고 분석하는 Skill이다.

**기능:**
- PDF 텍스트 추출
- 페이지별 내용 분석
- 이미지/차트 해석
- 문서 요약

```
사용자: report.pdf 파일의 3페이지 내용 알려줘
Claude: [pdf Skill 활성화]
        3페이지 내용:
        ...
```

**활용 예시:**
- 보고서 요약
- 계약서 검토
- 논문 분석
- 매뉴얼 참조

### 3-2 xlsx (엑셀 처리)

엑셀 및 스프레드시트 파일을 처리하는 Skill이다.

**기능:**
- 셀 데이터 읽기
- 시트별 분석
- 데이터 요약
- 수식 해석

```
사용자: sales.xlsx에서 총 매출 계산해줘
Claude: [xlsx Skill 활성화]
        매출 데이터를 분석합니다...

        총 매출: 1,234,567원
```

**활용 예시:**
- 데이터 분석
- 보고서 생성
- 수치 검증
- 차트 데이터 추출

### 3-3 ms-office-suite (오피스 통합)

Microsoft Office 문서 전체를 처리하는 통합 Skill이다.

**지원 형식:**
- .docx (Word)
- .xlsx (Excel)
- .pptx (PowerPoint)

```
사용자: presentation.pptx 슬라이드 내용 정리해줘
Claude: [ms-office-suite Skill 활성화]
        슬라이드 내용:

        슬라이드 1: 제목
        슬라이드 2: 개요
        ...
```

## 4. 사용자 정의 Skills

### 4-1 Skills 파일 위치

프로젝트 루트에 `.claude/skills/` 폴더를 만들어 사용한다.

```
프로젝트/
└── .claude/
    └── skills/
        └── skill이름.md
```

### 4-2 Skills 파일 구조

마크다운 파일로 Skill의 동작을 정의한다.

```markdown
# skill이름.md

이 Skill은 [목적]을 위해 사용됩니다.

## 활성화 조건
- [조건 1]
- [조건 2]

## 수행 작업
1. [작업 1]
2. [작업 2]
3. [작업 3]

## 출력 형식
[결과 형식 설명]
```

### 4-3 작성 예시

**코드 리뷰 Skill:**
```markdown
# code-review.md

코드 품질을 검토하고 개선점을 제안합니다.

## 활성화 조건
- 코드 파일이 포함된 리뷰 요청
- "리뷰", "검토" 키워드 포함

## 수행 작업
1. 코드 구조 분석
2. 보안 취약점 확인
3. 성능 문제 검토
4. 모범 사례 준수 확인

## 출력 형식
- 요약: 전체 평가
- 문제점: 발견된 이슈
- 제안: 개선 방안
```

**데이터 분석 Skill:**
```markdown
# data-analysis.md

데이터를 분석하고 인사이트를 도출합니다.

## 활성화 조건
- CSV, JSON, 엑셀 파일 분석 요청
- "분석", "통계" 키워드 포함

## 수행 작업
1. 데이터 구조 파악
2. 기초 통계 계산
3. 이상치 탐지
4. 트렌드 분석

## 출력 형식
- 데이터 개요
- 주요 통계
- 발견된 패턴
- 권장 사항
```

## 5. 실전 활용

### 5-1 문서 분석 워크플로우

```
사용자: 이 계약서 PDF 검토하고 주요 조항 정리해줘

Claude:
1. [pdf Skill 활성화]
2. PDF 내용 추출
3. 조항별 분석
4. 주요 내용 정리

결과:
계약서 주요 조항:
- 계약 기간: 2025.01.01 ~ 2025.12.31
- 계약 금액: 월 100만원
- 해지 조건: 30일 전 서면 통지
...
```

### 5-2 데이터 처리 자동화

```
사용자: 매출 엑셀 파일에서 월별 추이 분석해줘

Claude:
1. [xlsx Skill 활성화]
2. 매출 데이터 읽기
3. 월별 집계
4. 추이 분석

결과:
월별 매출 추이:
- 1월: 1,000만원
- 2월: 1,200만원 (+20%)
- 3월: 1,500만원 (+25%)
...
전체 추세: 상승세
```

### 5-3 Skills 조합 활용

여러 Skill을 조합해서 복잡한 작업을 처리할 수 있다.

```
사용자: PDF 보고서와 엑셀 데이터를 비교 분석해줘

Claude:
1. [pdf Skill] 보고서 내용 추출
2. [xlsx Skill] 엑셀 데이터 분석
3. 두 데이터 비교
4. 불일치 항목 식별

결과:
비교 분석 결과:
- 일치: 45개 항목
- 불일치: 3개 항목
  - 항목 A: 보고서 100, 엑셀 95
  ...
```

## 6. 정리

### Skills 요약

| Skill | 용도 | 지원 형식 |
|-------|------|----------|
| pdf | PDF 문서 처리 | .pdf |
| xlsx | 스프레드시트 처리 | .xlsx, .xls |
| ms-office-suite | 오피스 통합 | .docx, .xlsx, .pptx |

### Commands와 비교

| 구분 | Commands | Skills |
|------|----------|--------|
| 호출 | /명령어 | 자동 또는 Skill 도구 |
| 목적 | 즉시 동작 실행 | 파일/작업 전문 처리 |
| 정의 위치 | .claude/commands/ | .claude/skills/ |
| 사용자 정의 | 가능 | 가능 |

### 사용자 정의 Skills 요약

```
위치: .claude/skills/skill이름.md
내용: 활성화 조건, 수행 작업, 출력 형식
활용: 반복 작업 자동화, 전문 처리 표준화
```
