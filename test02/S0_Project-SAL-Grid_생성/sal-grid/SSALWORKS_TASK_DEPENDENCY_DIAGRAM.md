## 핵심 경로 (Critical Path)

프로젝트 완료까지 가장 긴 의존성 체인:

```
S1F1 → S1BI1 → S1S1 → S2BA1 → S2S1 → S3S1
                              ↓
                           S2BA3 → S4BA1 → S4BA2 → S4T2 → S5O1 → S5O2 → S5O3
```

**핵심 경로 Task 목록:**
1. S1F1 (Vercel 프로젝트 설정)
2. S1BI1 (환경변수 설정)
3. S1S1 (Supabase Auth Provider 설정)
4. S2BA1 (Google OAuth Serverless API)
5. S2BA3 (구독 관리 API)
6. S4BA1 (결제 API)
7. S4BA2 (결제 웹훅 API)
8. S4T2 (API 통합 테스트)
9. S5O1 (프로덕션 배포)
10. S5O2 (도메인 연결)
11. S5O3 (SSL 인증서 확인)
