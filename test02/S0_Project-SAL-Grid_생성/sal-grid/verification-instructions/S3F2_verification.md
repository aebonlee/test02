## 검증 항목

### 1. iframe 제거 확인
- [ ] `index.html`에서 aitalker.co.kr iframe 코드 제거됨
- [ ] 기존 `#ai-tutor-popup` 요소 제거됨

### 2. 새 모달 UI 확인
- [ ] `#ai-tutor-modal` 요소 존재
- [ ] 헤더 (새 대화, 크레딧, 닫기 버튼)
- [ ] 메시지 영역
- [ ] 입력 영역 (textarea + 전송 버튼)
- [ ] 사이드바 (대화 목록)

### 3. 기능 테스트

#### 3-1. 모달 열기/닫기
- [ ] AI 튜터 버튼 클릭 시 모달 열림
- [ ] 닫기 버튼 클릭 시 모달 닫힘
- [ ] ESC 키로 모달 닫힘 (선택사항)

#### 3-2. 메시지 전송
- [ ] 텍스트 입력 후 전송 버튼 클릭
- [ ] Enter 키로 전송 (Shift+Enter는 줄바꿈)
- [ ] 전송 후 입력창 초기화

#### 3-3. 스트리밍 렌더링
- [ ] 사용자 메시지 즉시 표시
- [ ] AI 응답 실시간 타이핑 효과
- [ ] 스크롤 자동 이동 (새 메시지)

#### 3-4. 참조 문서 표시
- [ ] AI 응답 하단에 참조 문서 표시
- [ ] 문서 제목 클릭 가능 (선택사항)

#### 3-5. 대화 히스토리
- [ ] 사이드바 토글 버튼 동작
- [ ] 대화 목록 표시
- [ ] 대화 클릭 시 해당 대화 로드
- [ ] 새 대화 버튼 동작

#### 3-6. 크레딧 표시
- [ ] 현재 크레딧 잔액 표시
- [ ] 질문 후 크레딧 업데이트

### 4. 반응형 디자인

#### 4-1. 데스크톱 (> 768px)
- [ ] 모달 너비 400px
- [ ] 우측 사이드 패널 형태

#### 4-2. 모바일 (< 768px)
- [ ] 모달 전체 화면
- [ ] 입력창 하단 고정
- [ ] 터치 스크롤 정상 동작

### 5. 에러 처리
- [ ] 네트워크 에러 시 에러 메시지 표시
- [ ] 크레딧 부족 시 안내 메시지
- [ ] Rate limit 초과 시 안내 메시지

### 6. 빌더 계정 체크
- [ ] 비빌더 계정은 AI 튜터 접근 불가
- [ ] 빌더 계정만 AI 튜터 사용 가능
