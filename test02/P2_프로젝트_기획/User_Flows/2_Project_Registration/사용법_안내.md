## 3️⃣ Order Sheet로 작업 지시

### Order Sheet란?

Claude Code에게 작업을 지시하는 **표준화된 문서**입니다.

### 작업 흐름

```
1. Dashboard에서 작업 단계 선택
   ↓
2. "Order Sheet 발행" 버튼 클릭
   ↓
3. 템플릿이 자동으로 로드됨
   ↓
4. 작업 내용 작성 및 수정
   ↓
5. "Download Order Sheet" 클릭
   ↓
6. Inbox/ 폴더에 자동 저장
   ↓
7. Claude Code 실행 (터미널에서 `claude` 입력)
   ↓
8. Claude Code가 Inbox/ 모니터링
   ↓
9. 작업 수행
   ↓
10. Outbox/에 결과 저장
   ↓
11. Dashboard에서 "Load from Outbox" 클릭하여 확인
```

### Order Sheet 작성 팁

**명확한 목표 설정:**
- ❌ "로그인 기능 만들어주세요"
- ✅ "JWT 기반 로그인 기능을 구현해주세요. 토큰 만료 시간은 24시간, 리프레시 토큰 포함"

**구체적인 기술 스택 명시:**
- ❌ "API 만들어주세요"
- ✅ "Express.js로 REST API를 만들어주세요. TypeScript 사용, 에러 핸들링 미들웨어 포함"

**예상 결과물 명시:**
- 생성될 파일 목록
- 수정될 파일 목록
- 테스트 코드 포함 여부

## 4️⃣ Claude Code 실행 방법

### 방법 1: 작업 폴더에서 직접 실행 (권장)

1. 프로젝트 폴더로 이동
2. 폴더 내에서 **마우스 우클릭**
3. "터미널에서 열기" 또는 "Git Bash Here" 선택
4. 터미널에서 다음 명령어 실행:
   ```bash
   claude
   ```

### 방법 2: VS Code에서 실행

1. VS Code에서 프로젝트 폴더 열기
2. 터미널 열기 (`Ctrl + \`` 또는 `Cmd + \``)
3. 터미널에서 `claude` 명령어 실행

## 5️⃣ Inbox/Outbox 시스템

### Inbox/ 폴더
- Dashboard에서 작성한 Order Sheet가 자동 저장됩니다
- Claude Code가 이 폴더를 모니터링하여 새 작업을 감지합니다
- 파일 형식: `ordersheet_YYYY-MM-DD_HH-mm-ss.json`

### Outbox/ 폴더
- Claude Code가 작업 완료 후 결과를 저장합니다
- 코드, 문서, 분석 결과 등이 포함됩니다
- "Load from Outbox" 버튼으로 결과를 Dashboard에서 확인 가능
