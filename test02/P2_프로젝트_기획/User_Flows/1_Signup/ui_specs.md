#### 필드 상세

**1) 이메일**
- **라벨:** 이메일 *
- **플레이스홀더:** example@email.com
- **타입:** email
- **필수:** Yes
- **검증:**
  - 이메일 형식 (`@` 포함)
  - 중복 확인 (실시간)
- **높이:** 48px
- **테두리:** 1px solid #D1D5DB (기본), #10B981 (포커스)

**2) 비밀번호**
- **라벨:** 비밀번호 *
- **플레이스홀더:** 8자 이상 입력해주세요
- **타입:** password
- **필수:** Yes
- **검증:**
  - 최소 8자
  - 영문 + 숫자 + 특수문자 포함
- **도움말:** ℹ️ 8자 이상, 영문+숫자+특수문자 포함
- **높이:** 48px
- **Show/Hide 버튼:** 우측에 눈 아이콘 (클릭 시 토글)

**3) 비밀번호 확인**
- **라벨:** 비밀번호 확인 *
- **플레이스홀더:** 비밀번호를 다시 입력해주세요
- **타입:** password
- **필수:** Yes
- **검증:** 비밀번호와 일치
- **높이:** 48px

**4) 닉네임**
- **라벨:** 닉네임 *
- **플레이스홀더:** 써니
- **타입:** text
- **필수:** Yes
- **검증:**
  - 2-20자
  - 중복 확인 (실시간)
- **도움말:** ℹ️ 2-20자, 다른 사용자에게 표시됨
- **높이:** 48px

**5) 실명**
- **라벨:** 실명 *
- **플레이스홀더:** 홍길동
- **타입:** text
- **필수:** Yes
- **검증:**
  - 2-50자
  - 한글 또는 영문
- **도움말:** ℹ️ 설치비 입금 시 본인 확인에 사용
- **높이:** 48px
