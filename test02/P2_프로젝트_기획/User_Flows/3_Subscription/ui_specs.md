- 카드 너비: 90%

- 카드 너비: 100%
- 좌우 패딩: 16px

### 1-5. 인터랙션

#### 결제 수단 선택
- 라디오 버튼 클릭 시 해당 섹션만 활성화
- 비활성화된 섹션: opacity 0.4
- 활성화 애니메이션: 200ms ease-in-out

#### 입력 필드 검증
- **계좌번호:** 숫자만 입력 가능, 10~14자리
- **카드번호:** 자동 하이픈 삽입 (4자리마다)
- **유효기간:** MM/YY 형식 자동 포맷
- **CVC:** 숫자 3~4자리
- 실시간 유효성 검사 표시
