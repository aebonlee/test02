# Font Size Rules (폰트 크기 규칙)

> 작성일: 2025-12-24
> 버전: v1.0
> 기준: WCAG 2.1 접근성 가이드라인 + 웹 표준

---

## 1. 기본 원칙

### 접근성 최소 기준
- **본문 텍스트**: 최소 14px (권장 16px)
- **보조 텍스트**: 최소 12px
- **읽어야 하는 모든 텍스트**: 최소 12px
- **뱃지/태그**: 최소 11px
- **장식 요소**: 최소 10px

### 금지 사항
- ❌ 10px 미만 폰트 사용 금지 (장식 포함)
- ❌ 읽어야 하는 텍스트에 11px 이하 사용 금지
- ❌ 본문/제목에 12px 미만 사용 금지

---

## 2. 8단계 폰트 스케일

| Level | Size | Name | 용도 | CSS Variable |
|-------|------|------|------|--------------|
| 1 | **28px** | Display | 대형 타이틀, 히어로 | `--font-display` |
| 2 | **24px** | H1 | 페이지 제목 | `--font-h1` |
| 3 | **20px** | H2 | 섹션 제목 | `--font-h2` |
| 4 | **18px** | H3 | 서브섹션 제목 | `--font-h3` |
| 5 | **16px** | H4 | 카드 제목, 강조 본문 | `--font-h4` |
| 6 | **14px** | Body | 기본 본문, 버튼 | `--font-body` |
| 7 | **13px** | Small | 보조 텍스트, 설명 | `--font-small` |
| 8 | **12px** | Caption | 캡션, 힌트, 날짜 | `--font-caption` |
| - | **11px** | Tag | 뱃지, 태그 전용 | `--font-tag` |
| - | **10px** | Icon | 장식 아이콘 전용 | `--font-icon` |

| 요소 | 이전 | 이후 | 이유 |
|------|------|------|------|
| 모달 제목 | 18-20px | **18px** | H3 |
| 모달 본문 | 14px | **14px** | Body |

| 요소 | 이전 | 이후 | 이유 |
|------|------|------|------|

## 4. 마이그레이션 맵

### 삭제 대상 (규격 외)
```
5px  → 10px (장식) 또는 삭제
7px  → 10px (장식)
9px  → 11px (뱃지) 또는 12px (텍스트)
```

### 상향 조정 대상
```
10px (읽어야 하는 텍스트) → 12px
11px (중요 텍스트)        → 13px
11px (보조 텍스트)        → 12px
```

### 유지 대상
```
10px - 알림 뱃지 숫자, 장식 아이콘
11px - 상태 뱃지, 태그
12px - 캡션, 날짜, 힌트
13px - 보조 설명
14px - 본문, 버튼
16-28px - 제목류
```

## 5. CSS Variables

```css

    /* Font Size Scale */
    --font-display: 28px;
    --font-h1: 24px;
    --font-h2: 20px;
    --font-h3: 18px;
    --font-h4: 16px;
    --font-body: 14px;
    --font-small: 13px;
    --font-caption: 12px;
    --font-tag: 11px;
    --font-icon: 10px;
}
```

## 6. 변경 요약

| 변경 유형 | 개수 | 내용 |
|----------|------|------|
| 5px → 10px | 2 | 불릿 기호 |
| 7px → 10px | 5 | 화살표, 불릿 |
| 9px → 12px | 3 | 퍼센트, 텍스트 |
| 10px → 12px | ~5 | 날짜, 라벨 등 |
| 11px → 13px | ~40 | 프로세스명, 프로젝트명 등 |
| 11px → 12px | ~30 | 퍼센트, Task ID 등 |
| **총 변경** | **~85개** | |
