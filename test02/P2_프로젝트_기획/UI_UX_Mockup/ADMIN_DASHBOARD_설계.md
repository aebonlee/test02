## 1. 핵심 기능

### 1.1 사용자 문의 관리
- 📬 **실시간 문의 모니터링**
  - 새 문의 알림
  - 미답변 문의 카운트
  - 문의 우선순위 표시

- ✍️ **답변 작성 및 전송**
  - 마크다운 에디터
  - 답변 템플릿
  - 실시간 미리보기

- 📊 **문의 통계**
  - 일별/주별/월별 문의 현황
  - 문의 유형별 분류
  - 평균 응답 시간

### 1.2 사용자 관리
- 👥 **사용자 목록**
  - 전체 사용자 조회
  - 활성/비활성 상태
  - 가입일, 마지막 활동

- 🔍 **사용자 검색**
  - 이메일/이름으로 검색
  - 필터링 (활성/비활성, 가입 기간 등)

- 📈 **사용자 활동**
  - Order 작성 이력
  - Perplexity 사용량
  - 문의 히스토리

### 1.3 시스템 모니터링
- 📊 **대시보드 개요**
  - 오늘의 통계 (신규 가입, 활성 사용자, 문의 건수)
  - 서버 상태
  - API 사용량 (Perplexity)

- 🔔 **알림 센터**
  - 새 문의 알림
  - 시스템 오류 알림
  - 중요 이벤트

### 1.4 콘텐츠 관리
- 📚 **학습용 콘텐츠 관리**
  - 콘텐츠 추가/수정/삭제
  - 카테고리 관리

- 📢 **공지사항 관리**
  - 공지사항 작성
  - 사용자 대시보드에 표시

## 4. UI/UX 디자인 원칙

### 4.1 색상 체계

**Primary Colors:**
- 관리자 전용 색상: `#4a5568` (다크 그레이)
- 강조 색상: `#f59e0b` (오렌지) - 알림, 중요 액션
- 성공: `#10b981` (그린)
- 경고: `#f59e0b` (오렌지)
- 위험: `#ef4444` (레드)

**사용자 대시보드와 차별화:**
- 사용자: 보라색 그라데이션 (#667eea → #764ba2)
- 관리자: 다크 톤 + 오렌지 포인트

### 4.2 디자인 스타일

**모던 & 프로페셔널:**
- 깔끔한 카드 기반 레이아웃
- 넉넉한 여백
- 부드러운 그림자
- 명확한 계층 구조

**정보 밀도:**
- 한 눈에 파악 가능한 통계
- 중요 정보 우선 배치
- 불필요한 장식 최소화

### 4.3 인터랙션

**즉각적인 피드백:**
- 버튼 클릭 시 로딩 표시
- 성공/실패 토스트 알림
- 실시간 데이터 업데이트

**효율적인 워크플로우:**
- 키보드 단축키 지원
- 빠른 액션 버튼
- 드래그 앤 드롭

## 5. 기술 스택

### 5.1 Frontend
- **HTML5 + CSS3**
- **Vanilla JavaScript** (프로토타입)
- 향후: React/Vue (필요시)

### 5.2 Backend
- **Node.js + Express**
- 기존 inbox_server.js 확장

### 5.3 Database (향후)
- **Supabase** (사용자, 문의 데이터)
- PostgreSQL 기반

### 5.4 실시간 통신
- **WebSocket** (실시간 알림)
- **Server-Sent Events** (SSE)
