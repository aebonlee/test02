## Dashboard Overview (홈)

### 주요 통계 카드 (4개)
1. **총 회원 수** (보라색) - 전일 대비 증감 표시
2. **이번 달 매출** (초록색) - 전월 대비 증감 표시
3. **크레딧 판매** (노란색) - 전월 대비 증감 표시
4. **미처리 문의** (빨간색) - 긴급 알림 표시

### 최근 활동 타임라인
- 실시간 업데이트
- 색상별 상태 표시 (🔴🟢🟡)
- 상대 시간 표시

### 긴급 처리 필요
- 우선순위별 작업 목록
- 클릭 시 해당 섹션 이동
- 건수 표시

### 차트 (2개)
- 회원 가입 추이 (Line Chart, 7일)
- 매출 추이 (Bar Chart, 4주)
- Chart.js 4.4.0 사용

### 최근 문의/가입 사용자
- 최신 5건 표시
- "전체 보기" 버튼

## Main Features

### 공지사항 관리
- CRUD 작업
- 중요도 표시 (🔴 중요 / 일반)
- 테이블 형식
- 실시간 업데이트 (Supabase)

### 학습용 콘텐츠 관리
- 카테고리별 관리
- 아티클 CRUD
- Markdown 에디터
- 공개/비공개 설정

### FAQ 관리
- 질문/답변 CRUD
- 카테고리별 분류
- 조회수 통계

### Order Sheet 템플릿
- 템플릿 생성/수정
- Markdown 형식
- 사용 통계

### 회원 관리
- 회원 목록 테이블
- 검색/필터링
- 구독 상태 확인
- 크레딧 잔액 확인

### 구독 관리
- 구독 플랜별 통계
- 미납 회원 알림
- 구독 취소 처리

### 결제 관리
- 결제 내역 목록
- 환불 처리
- 입금 확인 (무통장 입금)

### 크레딧 관리
- 크레딧 판매 내역
- 수동 지급/차감
- 환불 요청 처리

### 문의 관리
- 문의 목록
- 답변 작성
- 상태 변경 (미답변/답변완료)
- 파일 첨부 확인

---

## Technical Stack
- Frontend: Vanilla JavaScript (ES6+)
- Backend: Supabase (PostgreSQL + Realtime)
- Charts: Chart.js 4.4.0
- Security: DOMPurify 3.0.6 (XSS 방지)
