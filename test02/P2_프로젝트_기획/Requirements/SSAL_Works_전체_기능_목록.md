## 주요 사용자 플로우

### 1. 신규 사용자 가입 플로우
```
회원가입 → 이메일 인증 → 로그인 → 설치비 결제 → 관리자 승인 → 빌더 계정 활성화 → 웰컴 크레딧 지급
```

### 2. AI 개발 협업 플로우
```
프로젝트 생성 → SAL Grid 확인 → Order Sheet 작성 → Claude Code 전달 → 개발 진행 → 결과 확인
```

### 3. 크레딧 충전 플로우
```
크레딧 관리 → 충전 패키지 선택 → 결제 수단 선택 → 결제 완료 → 크레딧 즉시 반영
```

### 4. 학습 플로우
```
학습용 Books 열람 → Tips 확인 → AI에게 질문 → Sunny에게 문의 (필요시)
```
