## 개요

### 목적
SSAL Works의 모든 User Flow가 실제 사용자 여정 속에서 어떻게 연결되고 작동하는지 보여줌

### 4대 User Flow 통합
1. **User Flow #1:** 회원가입 및 온보딩
2. **User Flow #2:** 프로젝트 등록 및 관리
3. **User Flow #3:** 구독 관리 (설치비 + 월 사용료)
4. **User Flow #4:** AI 크레딧 충전 및 사용

### 사용자 유형별 여정
- **신규 사용자:** 가입 → 온보딩 → 첫 프로젝트 → 구독 시작
- **활성 사용자:** AI 활용 → 크레딧 관리 → 프로젝트 성장
- **챌린지 참여자:** 3개월 목표 달성 → 환불 신청
- **장기 사용자:** 지속 사용 → 추가 프로젝트 → 성공 사례


| 금액 | 영어 | 코드 |

| 300만원 | THREE | TH |
| 400만원 | FOUR | FO |
| 500만원 | FIVE | FI |
| 600만원 | SIX | SI |
| 700만원 | SEVEN | SE |
| 800만원 | EIGHT | EI |
| 900만원 | NINE | NI |

**예시:**
- 2025년 12월, 첫 번째 가입자, 300만원 납부 → `2512000001TH`
- 2026년 1월, 세 번째 가입자, 500만원 납부 → `2601000003FI`
