# SSAL Works가 제공하는 가치

> **최종 업데이트:** 2025-12-23
> **용도:** 서비스 소개, 마케팅

---

## SSAL Works만의 고유한 가치

### 1. 혼자서 웹사이트를 만들 수 있다

개발팀이 필요 없다.
개발자를 구하느라 시간과 에너지를 쓰지 않아도 된다.
수천만 원의 개발 비용이 필요 없다.
외주 개발 계약도, 의사소통 비용도 없다.
AI가 코딩을 담당하고, 본인은 기획과 의사결정만 하면 된다.
원하는 것을 정확히 만들 수 있다.

---

### 2. 2,500시간의 시행착오를 줄일 수 있다

2,500시간 안에는:
- 수많은 공부 (웹개발 기초부터 고급 기술까지)
- 수많은 AI 사용 경험 (Claude, ChatGPT, Gemini, Perplexity...)
- 수많은 좌절 (안 되는 것, 막히는 것, 모르는 것의 연속)
- 수많은 극복 (포기하지 않고 해결책을 찾음)
- 수많은 시행착오 (잘못된 방향으로 며칠씩 헤맴)

이 모든 것이 들어있다.

한 사람이 이 모든 시행착오를 겪어서 다 알려주면,
다른 사람들은 시행착오 없이 바로 할 수 있다.
쓸데없는 시행착오 겪지 말고 바로 시작할 수 있다.

---

### 3. 표준 디렉토리 구조를 제공한다

AI를 방치하면 여기저기에 폴더와 파일을 만든다.
제비가 사방에 제비집을 짓는 것처럼 엉망이 된다.

SSAL Works는 검증된 표준 디렉토리 구조를 제공한다:
- P0~P3 (준비 단계)
- S0~S5 (실행 단계)
- Production (배포용 코드)
- .claude (AI 관리 설정)
- Human_ClaudeCode_Bridge (인간-AI 소통 채널)

---

### 4. AI에게 체계적으로 작업을 지시하는 Order Sheet 체계를 제공한다

AI에게 "이거 해줘"라고만 하면 엉터리가 된다.
체계화된 작업 지시서(Order Sheet)가 필요하다.

SSAL Works가 제공하는 Order Sheet 체계:
- Part A: 모든 프로젝트에 공통으로 적용되는 표준 규칙
- Part B: 프로젝트별로 커스터마이징하는 부분
- 7단계 작업 순서: 이해 → 검증 → 규칙확인 → 질문 → 실행 → Stage Gate → PO승인
- AI 준수 사항, 파일 저장 규칙, SAL Grid 기록 규칙

---

### 5. 개발 단계별 상세한 안내문을 제공한다

각 개발 단계마다 무엇을 해야 하는지 상세히 안내한다:
- 회원가입 후 안내
- 프로젝트 등록 후 사용법 안내
- 각 단계별 Order Sheet 템플릿
- 환영 메시지, 작성법 가이드

뭘 해야 할지 몰라서 멍하니 있는 시간이 없다.

---

### 6. AI가 지켜야 할 규칙 파일을 제공한다

AI가 제멋대로 하지 않도록 7개의 규칙 파일을 제공한다:
- 01_file-naming.md (파일 명명 규칙)
- 02_save-location.md (저장 위치 규칙)
- 03_area-stage.md (11개 Area, 5개 Stage 매핑)
- 04_grid-writing-supabase.md (Grid 기록 규칙)
- 05_execution-process.md (6단계 실행 프로세스)
- 06_verification.md (Task/Stage Gate 검증 기준)
- 07_task-crud.md (Task 추가/삭제/수정 규칙)

---

### 7. SAL Grid 기반 프로젝트 관리 방법론을 제공한다

3차원 좌표계(Stage, Area, Level)로 프로젝트 전체를 관리한다:
- Stage: 개발 진행 단계 (S1~S5)
- Area: 개발 분야 (11개 영역)
- Level: 개별 Task 순서
- 22개 Task 속성으로 진행 상황 추적
- Task Instruction / Verification Instruction
- Stage Gate 검증

**회계감사 경험이 녹아있는 방법론:**

| 회계감사 개념 | SAL Grid 적용 |
|--------------|--------------|
| 감사 계획서 | Order Sheet |
| 감사 절차서 | Task Instruction |
| 내부통제 테스트 | Verification Instruction |
| Working Paper | Task Results |
| Review Process | Stage Gate 검증 |
| 감사조서 보관 | 이중 저장 |
| 파트너 승인 | PO 승인 |

**특히 강점:**
- 작업자와 검증자 분리 (Preparer/Reviewer 분리)
- 모든 작업의 추적 가능성 (Audit Trail)
- 단계별 Gate 승인 (품질 통제)
- 문서화 철저 (증빙 확보)

비개발자가 만들었지만, 개발자들이 놓치기 쉬운 **관리/통제/검증** 측면이 탄탄하다.

### 9. 외부 연동 설정 가이드를 제공한다

외부 서비스 연동은 AI가 할 수 없다.
사람이 직접 해야 하는데, 굉장히 어렵고 복잡하고 시간이 많이 걸린다.

SSAL Works는 이미 성공한 외부 연동 설정 가이드 5가지를 제공한다:
- **Supabase** (데이터베이스, 인증 기반)
- **Google OAuth** (소셜 로그인)
- **Resend** (이메일 발송)
- **Vercel** (배포, 도메인 연결)
- **토스페이먼츠** (결제 시스템)

이 가이드를 따라하면 작업 시간이 획기적으로 줄어든다.
처음부터 삽질하지 않아도 된다.

---

### 10. ChatGPT, Gemini, Perplexity를 별도 구독 없이 사용할 수 있다

SSAL Works 내에서 ChatGPT, Gemini, Perplexity를 바로 사용할 수 있다.
각각 별도로 유료 구독하지 않아도 된다.
크레딧 차감 방식으로 필요할 때만 사용.

---

### 11. 웹사이트 개발부터 비즈니스 실현까지 함께한다

**SSAL Works = 컴퍼니빌더형 액셀러레이터**

웹사이트 개발에 참여하고, 후속적인 지원을 합니다.

```
웹사이트 개발 (수단)
      ↓
비즈니스 실현 (목표)
```

단순히 웹사이트 개발만 하는 게 아닙니다.
웹사이트 개발을 통해서 비즈니스를 실현하는 **빌더**입니다.

**비즈니스가 실현될 때까지** SSAL Works가 함께합니다.

---

**3가지 지원 영역:**

**[1] 웹사이트 개발**
- 프로젝트 방향 설정
- 기술적 문제 해결
- 개발 과정 멘토링

**[2] 창업 멘토링**
- 사업 아이디어 검증
- 비즈니스 모델 수립
- 창업 전략 자문
- 투자 유치 준비
- 사업계획서 작성/리뷰
- 특허 출원 가이드 (직접 다수 출원 경험)

**[3] 회계/세무/경영 자문**
- 법인 설립
- 세금 신고
- 재무제표 이해
- 자금 관리
- 경영 전략 수립

---

공인회계사 + 10년 이상 스타트업 액셀러레이터 경험.

### 12. AI가 베낄 수 있는 실제 프로젝트 수십 개를 제공한다

AI가 가장 잘하는 것은 **베끼는 것**이다.
참고할 수 있는 실제 프로젝트가 많으면 많을수록, AI는 더 잘 만든다.

**현재 제공 (1개 프로젝트):**
- SSAL Works (본 플랫폼)

**조만간 추가 (2개 프로젝트):**
- Politician Finder (AI 기반 정치인 평가 플랫폼)
- HDH Fintech (핀테크 솔루션)

**내년 목표 (50개 프로젝트):**
- 약 50개의 풀스택 웹사이트를 추가 제작 예정
- 모든 프로젝트의 문서와 소스코드를 제공

**베낄 수 있는 것이 수십 개가 있으면, 어떤 웹사이트를 못 만들겠는가?**

바보가 아닌 한, 말만 할 수 있고 Claude Code만 켜서 대화만 할 수 있으면,
누구든지 웹사이트를 만들 수 있다.

SSAL Works는 **풀스택 웹사이트 개발 전문 미니 GitHub**가 된다.

빌더 계정 개설자에게 제공.
