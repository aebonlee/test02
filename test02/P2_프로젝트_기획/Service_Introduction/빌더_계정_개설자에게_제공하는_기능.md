# 빌더 계정 개설자에게 제공하는 기능

> **SSAL Works 플랫폼이 빌더 계정 개설자에게 제공하는 서비스 및 기능 상세**
>
> 최종 업데이트: 2025-12-23

---

## 개요

SSAL Works는 **"Claude Code를 활용한 3차원 SAL Grid 기반 풀스택 웹사이트 개발 실행 및 관리 플랫폼"**입니다. 빌더 계정 개설자(₩3,000,000 개설비 납부 완료자)에게 다음의 기능들을 제공합니다.

화면은 **진행 · 통제 · 관리 · 학습 · 질문** 5가지 영역으로 구성되어 있으며, 각 영역별로 제공되는 기능을 상세히 설명합니다.

---

## 1. 프로젝트 실행 및 관리 (좌측 사이드바)

좌측 사이드바에서 프로젝트를 등록하고 실행 과정을 관리하는 기능을 제공합니다.

### 1-1. PROJECT 섹션

1. **예시 Project**: SSAL Works가 제공하는 예시 프로젝트를 열람할 수 있습니다. 처음 플랫폼을 사용하는 빌더 계정 개설자가 프로젝트 진행 방법을 이해할 수 있도록 완성된 예시를 제공합니다.

2. **완료 Project**: 이미 완료된 프로젝트 목록을 확인할 수 있습니다. 과거에 진행했던 프로젝트들의 기록을 보관하고 참조할 수 있습니다.

3. **진행중 Project**: 현재 진행 중인 프로젝트를 선택하여 작업을 이어갈 수 있습니다. 프로젝트를 클릭하면 해당 프로젝트의 SAL Grid와 Order Sheet 작업 공간이 활성화됩니다.

4. **새로운 Project 등록**: 새 프로젝트를 생성할 수 있습니다. 프로젝트명과 설명을 입력하여 등록하면, 해당 프로젝트의 SAL Grid가 자동으로 생성됩니다.

### 1-2. 진행 프로세스 섹션

SSAL Works의 프로젝트 진행은 **예비단계(P0~P3)**와 **실행단계(S0~S5)**로 구분됩니다. 진행 프로세스 섹션에서 각 단계의 진행률과 상태를 한눈에 확인할 수 있습니다.

**예비단계:**
1. **P0. 작업 디렉터리 구조 생성**: 프로젝트의 기본 폴더 구조를 생성하는 단계입니다.
2. **P1. 사업계획**: 비즈니스 모델과 사업 계획을 수립하는 단계입니다.
3. **P2. 프로젝트 기획**: 개발할 서비스의 기능과 사용자 흐름을 기획하는 단계입니다.
4. **P3. 프로토타입 제작**: 실제 개발 전 프로토타입을 제작하는 단계입니다.

**실행단계:**
1. **S0. Project SAL Grid 생성**: 3차원 Grid 기반의 Task 관리 시스템을 생성하는 단계입니다.
2. **S1. 개발 준비**: 개발 환경 설정, 기초 세팅을 완료하는 단계입니다.
3. **S2. 개발 1차**: 핵심 기능을 개발하는 단계입니다.
4. **S3. 개발 2차**: 고급 기능을 개발하는 단계입니다.
5. **S4. 개발 3차**: 최적화와 테스트를 진행하는 단계입니다.
6. **S5. 운영**: 배포 및 유지보수를 진행하는 단계입니다.

각 단계별로 진행률이 퍼센트(%)로 표시되며, Stage Gate를 통과하면 다음 단계로 진행할 수 있습니다.

## 2. Order Sheet 발행 (중앙 상단 - Control Space)

중앙 상단의 Control Space에서 AI(Claude Code)에게 작업을 지시하기 위한 Order Sheet를 발행하는 기능을 제공합니다.

### 2-1. 표준 Order Sheet 템플릿 제공

1. **Order Sheet 템플릿**: AI에게 작업을 요청하기 위한 표준화된 템플릿을 제공합니다. 템플릿에는 작업 유형, 작업 내용, 참고 자료, 기대 결과물 등의 항목이 포함되어 있어, 빌더 계정 개설자가 명확하게 작업을 지시할 수 있습니다.

2. **Order Sheet 작성 공간**: Control Space의 에디터에서 Order Sheet를 직접 작성하거나 수정할 수 있습니다. 마크다운 형식을 지원하며, 실시간으로 내용을 편집할 수 있습니다.

3. **Order Sheet 복사 기능**: 작성된 Order Sheet를 클립보드에 복사하여 Claude Code 세션에 바로 붙여넣기할 수 있습니다. "복사하기" 버튼을 클릭하면 Order Sheet 전체 내용이 복사됩니다.

### 2-2. AI에게 Order Sheet 전달

1. **Claude Code 연동**: Order Sheet를 복사한 후, Claude Code 터미널에 붙여넣기하면 AI가 Order Sheet를 읽고 작업을 시작합니다.

2. **작업 지시 전달 프로세스**:
   - 대시보드에서 Order Sheet 작성
   - "Order Sheet 복사하기" 버튼 클릭
   - Claude Code 터미널에 붙여넣기
   - AI가 Order 읽고 작업 수행

### 2-3. 결과 Report 수신

1. **작업 결과 확인**: AI가 작업을 완료하면 Human_ClaudeCode_Bridge/Reports 폴더에 결과 보고서가 저장됩니다.

2. **Reports 불러오기**: 대시보드의 "Reports 불러오기" 버튼을 클릭하면 Reports 폴더의 파일 목록을 확인하고, 원하는 Report 파일을 선택하여 내용을 열람할 수 있습니다.

3. **결과물 형식**: 결과 보고서는 JSON 형식으로 저장되어 AI 메모리 활용과 자동화에 최적화되어 있습니다. 필요시 마크다운(.md) 형식의 요약 문서도 함께 제공됩니다.

## 3. PROJECT SAL GRID를 통한 개발 과정 관리 (중앙 하단)

중앙 하단에서 PROJECT SAL GRID를 통해 개발 과정을 체계적으로 관리하는 기능을 제공합니다.

### 3-1. PROJECT SAL GRID 시스템

1. **3차원 Task 관리 시스템**: PROJECT SAL GRID는 3차원 좌표계(X-Y-Z)로 프로젝트의 모든 Task를 관리합니다.
   - **X축 (Stage)**: 개발 단계 (S1~S5)
   - **Y축 (Area)**: 개발 영역 (11개 분야)
   - **Z축 (Level)**: 개별 Task 단위

2. **11개 Area 분류**:
   - M (Documentation): 문서화
   - U (Design): UI/UX 디자인
   - F (Frontend): 프론트엔드
   - BI (Backend Infrastructure): 백엔드 기반
   - BA (Backend APIs): 백엔드 API
   - D (Database): 데이터베이스
   - S (Security): 보안/인증/인가
   - T (Testing): 테스트
   - O (DevOps): 운영/배포
   - E (External): 외부 연동
   - C (Content System): 콘텐츠 시스템

### 3-2. Grid 뷰어 기능

1. **2D Card View**: 각 Task를 카드 형태로 보여주는 2차원 뷰입니다. Task의 상태, 진행률, 담당 Agent 등의 정보를 직관적으로 확인할 수 있습니다.

2. **3D Block View**: Task들을 3차원 블록으로 시각화하여 전체 프로젝트 구조를 입체적으로 파악할 수 있는 뷰입니다.

3. **풀스크린 모드**: Grid를 전체 화면으로 확대하여 더 넓은 시야에서 프로젝트를 관리할 수 있습니다.

### 3-3. Task 상태 관리

1. **상태별 통계 표시**: Grid 상단에 각 상태별 Task 수가 실시간으로 표시됩니다.
   - **Pending (대기)**: 아직 시작되지 않은 Task
   - **In Progress (진행)**: 현재 작업 중인 Task
   - **Executed (실행)**: 실행이 완료된 Task
   - **Completed (완료)**: 검증까지 완료된 Task
   - **Fixing (수정)**: 수정이 필요한 Task

2. **Task 클릭 상세 정보**: 각 Task를 클릭하면 Task Instruction, 담당 Agent, 생성된 파일, 검증 결과 등의 상세 정보를 확인할 수 있습니다.

### 3-4. Stage Gate 검증

1. **Stage Gate 시스템**: 각 Stage가 완료되면 Stage Gate 검증을 통해 다음 Stage로 진행할 수 있는지 확인합니다.

2. **AI 검증 + PO 승인**: AI가 자동 검증을 수행한 후, Project Owner(빌더 계정 개설자)가 최종 승인하면 다음 Stage로 진행합니다.

### 3-5. PROJECT SAL GRID 매뉴얼

1. **Grid 매뉴얼 제공**: PROJECT SAL GRID의 사용 방법, 22개 속성 설명, Task 실행 및 검증 프로세스 등을 상세히 설명하는 매뉴얼을 제공합니다.

## 5. 외부 연동 설정 가이드 (5가지)

외부 서비스 연동은 AI가 혼자 할 수 없고, 빌더 계정 개설자(Human)와 AI가 함께 진행해야 하는 작업입니다. SSAL Works는 이 복잡한 과정을 단계별로 안내하는 **5가지 설정 가이드**를 제공합니다.

### 5-1. 🗄️ 데이터베이스 설정 (Supabase DB + RLS)

1. Supabase 프로젝트 생성 방법을 안내합니다.
2. 데이터베이스 테이블 생성 및 스키마 설정 방법을 설명합니다.
3. Row Level Security(RLS) 정책 설정으로 데이터 보안을 확보하는 방법을 다룹니다.
4. API 키 발급 및 환경 변수 설정 방법을 안내합니다.
5. Claude Code에서 Supabase를 활용하는 방법을 설명합니다.

### 5-2. 🔐 회원인증 설정 (Google OAuth + Supabase)

1. Google Cloud Console에서 OAuth 클라이언트 생성 방법을 안내합니다.
2. Supabase Auth Provider 설정 방법을 설명합니다.
3. 리디렉션 URL 구성 및 콜백 처리 방법을 다룹니다.
4. 로그인/로그아웃 플로우 구현 방법을 안내합니다.
5. 세션 관리 및 사용자 정보 활용 방법을 설명합니다.

### 5-3. 📧 이메일 시스템 설정 (Resend + SMTP)

1. Resend 계정 생성 및 API 키 발급 방법을 안내합니다.
2. 도메인 인증 및 발신 이메일 설정 방법을 설명합니다.
3. 이메일 템플릿 작성 및 발송 기능 구현 방법을 다룹니다.
4. 가입 환영 메일, 비밀번호 재설정 메일 등 자동 발송 설정을 안내합니다.
5. 이메일 발송 실패 처리 및 로깅 방법을 설명합니다.

### 5-4. 🚀 배포/도메인 설정 (Vercel + Whois)

1. Vercel 프로젝트 생성 및 GitHub 연동 방법을 안내합니다.
2. 환경 변수 설정 및 빌드 구성 방법을 설명합니다.
3. 도메인 구매 및 DNS 설정 방법(Whois 등)을 다룹니다.

## 7. 다른 AI에게 묻기 (ChatGPT · Gemini · Perplexity)

개발 중 궁금한 사항이 생겼을 때, 3개의 AI 서비스에 문의하여 답변을 받을 수 있는 기능을 제공합니다.

### 7-1. AI Q&A 서비스

1. **ChatGPT 문의**: OpenAI의 ChatGPT를 통해 코드 작성, 문서 생성, 일반적인 개발 질문에 대한 답변을 받을 수 있습니다.

2. **Gemini 문의**: Google의 Gemini를 통해 코드 리뷰, 설계 관련 질문에 대한 답변을 받을 수 있습니다.

3. **Perplexity 문의**: Perplexity를 통해 검색 기반의 정확한 정보와 최신 기술 동향에 대한 답변을 받을 수 있습니다.

### 7-2. 크레딧 시스템

1. **크레딧 차감 방식**: AI Q&A 서비스 이용 시 크레딧이 차감됩니다. 크레딧 잔액은 대시보드 상단에 실시간으로 표시됩니다.

2. **초기 크레딧 제공**: 빌더 계정 개설 시 ₩50,000 상당의 웰컴 크레딧이 제공됩니다.

3. **크레딧 충전**: 잔액이 부족할 경우 ₩10,000 / ₩20,000 / ₩30,000 / ₩50,000 단위로 추가 충전할 수 있습니다.

### 7-3. 질문/답변 이력 관리

1. **이력 저장**: 모든 질문과 답변은 자동으로 저장되어 나중에 다시 확인할 수 있습니다.

2. **이력 조회**: "질문/답변 이력" 버튼을 클릭하면 과거의 질문과 AI 답변 내역을 열람할 수 있습니다.

## 부가 기능

### 공지사항

우측 사이드바 상단에서 SSAL Works 플랫폼의 공지사항을 확인할 수 있습니다. 새로운 기능 출시, 시스템 점검 안내, 중요 업데이트 등의 정보가 게시됩니다.

### My Page

좌측 사이드바의 My Page 섹션에서 개인 정보 관리, 구독 상태 확인, 크레딧 관리 등을 할 수 있습니다.

1. **내 프로필**: 이름, 이메일, 빌더 계정 ID 조회 및 수정
2. **서비스 이용 현황**: 구독 상태, 다음 갱신일, 구독 취소
3. **크레딧 관리**: 잔액 조회, 사용 내역, 충전
4. **보안 설정**: 비밀번호 변경, 2단계 인증
5. **결제 수단 관리**: 카드 추가/삭제/변경

### 인앱 알림

대시보드 상단의 알림 아이콘을 통해 중요한 알림을 실시간으로 받아볼 수 있습니다.

1. **크레딧 잔액 부족 알림**: 크레딧이 일정 수준 이하로 떨어지면 알림
2. **크레딧 충전 완료 알림**: 충전이 완료되면 알림
3. **설치비 입금 확인 알림**: 입금이 확인되면 알림
4. **무료 기간 만료 임박 알림**: 플랫폼 이용료 무료 기간이 곧 만료됨을 알림
5. **시스템 공지 알림**: 중요한 시스템 공지를 알림
