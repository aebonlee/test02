## 6. 발견된 이슈 및 권장 조치

### 🔴 Critical (즉시 수정 필요)

**C1. console.log 민감 정보 노출**
- **파일:** `login.html` (L664), `signup.html` (L700), `admin-dashboard.html` 다수
- **문제:** 사용자 이메일, 인증 세션 정보가 브라우저 콘솔에 노출
- **예시:** `console.log('이미 로그인됨:', session.user.email);`
- **조치:** 프로덕션 배포 전 모든 console.log/error 제거 또는 조건부 실행
- **예상 소요:** 30분

### 🟡 High (빠른 수정 권장)

**H1. 로그인 페이지 XSS 방지 누락**
- **파일:** `login.html`
- **문제:** signup.html에는 DOMPurify가 적용되어 있으나, login.html에는 미적용
- **조치:** DOMPurify CDN 추가 및 이메일 입력값 sanitize 적용
- **예상 소요:** 15분

**H2. Empty 상태 개선 필요**
- **파일:** `index.html` (메인 대시보드)
- **문제:** 프로젝트가 없을 때 빈 화면으로 표시될 가능성
- **조치:** Empty 상태 UI 추가 (아이콘 + 안내 메시지 + "프로젝트 추가" CTA)
- **예상 소요:** 1시간

### 🟢 Medium (개선 권장)

**M1. 색상 대비 검증**
- **조치:** WebAIM Contrast Checker로 WCAG AA 기준 (4.5:1) 확인
- **대상:** --primary, --secondary, --danger 등 CSS 변수
- **예상 소요:** 30분

**M2. 환경변수 관리**
- **조치:** Supabase 키를 Vercel 환경변수로 관리
- **예상 소요:** 30분

**M3. 코드 주석 보완**
- **조치:** 복잡한 로직에 JSDoc 스타일 주석 추가
- **예상 소요:** 1시간
